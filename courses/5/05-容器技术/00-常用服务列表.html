使用bitnami版

bitnami帮你做了一些初始化，并且搭建集群模式也更加方便

具体配置查看: https://github.com/bitnami/bitnami-docker-mongodb

docker pull bitnami/mongodb:5.0

docker run -d -p 27017:27017 \
-e MONGODB_ROOT_PASSWORD=root123456 \
-e MONGODB_DATABASE=sys_db \
-e MONGODB_USERNAME=sys_user \
-e MONGODB_PASSWORD=password123 \
--name mongo bitnami/mongodb:5.0

然后，连接地址为DATABASE_URL=mongodb://sys_user:password123@192.168.0.102:27017/sys_db?authSource=sys_db

注意authSource表示着这个用户是在哪个数据库里面的

使用bitnami创建集群模式

  • MONGODB_ADVERTISED_HOSTNAME: 可以使用ip
docker run -d -p 27017:27017 \
-e MONGODB_REPLICA_SET_MODE=primary \
-e MONGODB_ADVERTISED_HOSTNAME=mongo \
-e MONGODB_REPLICA_SET_NAME=mongo \
-e MONGODB_REPLICA_SET_KEY=replicasetkey123 \
-e MONGODB_ENABLE_DIRECTORY_PER_DB=yes \
-e MONGODB_ROOT_PASSWORD=root123456 \
-e MONGODB_DATABASE=sys_db \
-e MONGODB_USERNAME=sys_user \
-e MONGODB_PASSWORD=password123 \
--name mongo bitnami/mongodb:5.0

连接地址为mongodb://sys_user:password123@192.168.0.102:27017/sys_db?authSource=sys_db&replicaSet=mongo&w=majority&directConnection=true

其中replicaSet就是MONGODB_REPLICA_SET_NAME

# 集群模式(TODO)

# 使用

etcdctl member list

# 加入数据
etcdctl put /testdir/testkey "Hello world"

etcdctl --user=admin:123456 put /mykey 'myvalue'
# 监听
etcdctl watch foo foo9

创建角色
• etcdctl role add root
• etcdctl role add read_write
• etcdctl role add read_only

• 为角色赋予权限（后面的prefix就是设定了只对哪部分有权限）
• etcdctl role grant-permission read_write --prefix=true readwrite /
• etcdctl role grant-permission read_only --prefix=true read /

• 创建用户
• etcdctl  user add root:123456
• etcdctl user add admin:123456
• etcdctl user add monitor:123456

• 用户与角色相绑定
• etcdctl user grant-role root root
• etcdctl user grant-role admin read_write
• etcdctl user grant-role monitor read_only

fluentd

docker pull fluent/fluentd:v1.14-debian

docker run -d -p 24224:24224  -v /home/fluentd/container-logs:/fluentd/log -v /home/fluentd/conf/fluent.conf:/fluentd/etc/fluent.conf fluent/fluentd:v1.14-debian

# 配置其他容器的日志，输出到fluentd中
# fluentd-async-connect：fluentd-docker异步设置,避免fluentd挂掉之后导致Docker容器也挂了
docker run -d --log-driver fluentd --log-opt fluentd-address=fluentd:24224 --log-opt tag="nginx-test" --log-opt fluentd-async-connect --name nginx-test -p 8080:80 nginx

配置文件示例

<source>
  @type   forward
</source>
<match *>
  @type              file
  path               /fluentd/log/${tag}/${tag}
  append             true
  <format>
    @type            single_value
    message_key      log
  </format>
  <buffer tag,time>
    @type             file
    timekey           1d
    timekey_wait      10m
    flush_mode        interval
    flush_interval    30s
  </buffer>
</match>
