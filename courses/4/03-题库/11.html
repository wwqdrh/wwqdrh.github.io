提示： 满足条件的箱子不超过1个。

key = input()
boxes = [x for x in input()[2:].split(" ")]

res = -1
for i in range(len(boxes)):
    lower = boxes[i].lower()
    char_set = set([])
    for c in lower:
        if 'a' <= c <= 'z':
            char_set.add(c)
    
    if len(char_set) == len(key):
        if key == "".join(char_set):
            res = i + 1
            break
print(res)

任务调度

现有一个CPU和一些任务需要处理，已提前获知每个任务的任务ID、优先级、所需执行时间和到达时间。 CPU同时只能运行一个任务，请编写一个任务调度程序，采用“可抢占优先权调度”调度算法进行任务调度，规则如下：

如果一个任务到来时，CPU是空闲的，则CPU可以运行该任务直到任务执行完毕。

但是如果运行中有一个更高优先级的任务到来，则CPU必须暂停当前任务去运行这个优先级更高的任务； 如果一个任务到来时，CPU正在运行一个比他优先级更高的任务时，信道大的任务必须等待； 当CPU空闲时，如果还有任务在等待，CPU会从这些任务中选择一个优先级最高的任务执行，相同优先级的任务选择到达时间最早的任务。

输入描述

输入有若干行，每一行有四个数字（均小于10^8）, 分别为任务ID，任务优先级，执行时间和到达时间。 每个任务的任务ID不同，优先级数字越大优先级越高， 并且相同优先级的任务不会同时到达。 输入的任务已按照到达时间从小到大排序，并且保证在任何时间， 处于等待的任务不超过10000个。

import functools
import sys

def comp(a, b):
    if a[1] > b[1]:
        return -1
    elif a[1] == b[1]:
        return 0
    else:
        return 1
    
def find(tasks, time):
    for task in tasks:
        if task[3] == time:
            return task
    return None

tasks = []
while True:
    line = sys.stdin.readline()
    if not line:
        break
    tasks.append([int(x) for x in line.split(" ")])

time = 0
waiting = []
while len(tasks) > 0:
    cur = find(tasks, time)
    if cur is not None:
        waiting.append(cur)
        waiting = sorted(waiting, key=functools.cmp_to_key(comp))
        cur = waiting[0]
    else:
        if len(waiting) != 0:
            cur = waiting[0]
    if cur is not None:
        cur[2] -= 1
        if cur[2] == 0:
            print(str(cur[0]) + " " + str(time + 1))
            tasks.remove(cur)
            waiting.remove(cur)
    time += 1

核酸检测安排

在系统、网络均正常的情况下组织核酸采样员和志愿者对人群进行核酸检测筛查。

每名采样员的效率不同，采样效率为N人/小时。

由于外界变化，采样员的效率会以M人/小时为粒度发生变化，M为采样效率浮动粒度，M=N*10%，输入保证N*10%的结果为整数。

采样员效率浮动规则：采样员需要一名志愿者协助组织才能发挥正常效率，在此基础上，每增加一名志愿者，效率提升1M，最多提升3M；

如果没有志愿者协助组织，效率下降2M。

怎么安排速度最快？求总最快检测效率（总检查效率为各采样人员效率值相加）。

def NucleicAcidTestArrangement(N: int, V: int, nums: list[int]) -> int:
    arrM = [0 for _ in range(N)]
    for i in range(N):
        arrM[i] = nums[i] / 10
    dp = [[0 for _ in range(V + 1)] for _ in range(N + 1)]
    count = 0
    for i in range(1, N + 1):
        count += (nums[i - 1] - 2 * arrM[i - 1])
        dp[i][0] = count
    
    for i in range(1, N+1):
        for j in range(1, V+1):
            dp[i][j] = max(dp[i-1][j] + nums[i-1] - 2 * arrM[i - 1], dp[i-1][j-1] + nums[i-1])
            dp[i][j] = max(dp[i][j], dp[i-1][j-2] + nums[i-1] + arrM[i-1] if j - 2 >= 0 else 0)
            dp[i][j] = max(dp[i][j], dp[i-1][j-3] + nums[i-1]+2*arrM[i-1] if j - 3 >= 0 else 0)
            dp[i][j] = max(dp[i][j], dp[i-1][j-4] + nums[i-1]+3+arrM[i-1] if j - 4 >= 0 else 0)
    return dp[N][V]

N, V = map(int, input().split())
nums = [int(x) for x in input().split(" ")]
print(NucleicAcidTestArrangement(N, V, nums))

计算网络信号

网络信号经过传递会逐层衰减，且遇到阻隔物无法直接穿透，在此情况下需要计算某个位置的网络信号值。 注意:网络信号可以绕过阻隔物。

array[m][n] 的二维数组代表网格地图， array[i][j] = 0代表i行j列是空旷位置; array[i][j] = x(x为正整数)代表i行j列是信号源，信号强度是x; array[i][j] = -1代表i行j列是阻隔物。 信号源只有1个，阻隔物可能有0个或多个 网络信号衰减是上下左右相邻的网格衰减1 现要求输出对应位置的网络信号值。

while True:

        m, n = map(int, input().split())
        data_list = list(map(int, input().split()))
        tx, ty = map(int, input().split())
        source = max(data_list)
        sx, sy = 0, 0
        array = []
        for i in range(m):
            tmp = data_list[i*n: i*n+m]
            if source in tmp:
                sx = i
                sy = tmp.index(source)
            array.append(tmp)
        
        def dfs(x, y, vol):
            if not 0 <= x < m or not 0 <= y < n:
                return
            if vol <= 0:
                return
            if array[x][y] == -1 or array[x][y] > vol:
                return
            
            array[x][y] = vol
            dfs(x+1, y, vol-1)
            dfs(x-1, y, vol-1)
            dfs(x, y+1, vol-1)
            dfs(x, y-1, vol-1)
        
        dfs(sx, sy, source)
        print(array[tx][ty])
    except Exception as e:
        break

星际篮球争霸赛

在星球争霸篮球赛对抗赛中，最大的宇宙战队希望每个人都能拿到MVP，MVP的条件是单场最高分得分获得者。 可以并列所以宇宙战队决定在比赛中尽可能让更多队员上场，并且让所有得分的选手得分都相同， 然而比赛过程中的每1分钟的得分都只能由某一个人包揽。

输入描述

输入第一行为一个数字 t ，表示为有得分的分钟数 1 ≤ t ≤ 50 第二行为 t 个数字，代表每一分钟的得分 p，1 ≤ p ≤ 50

输出描述

输出有得分的队员都是MVP时，最少得MVP得分。

def canPartitionKSubsets(nums, k):
    all = sum(nums)
    if all % k:
        return False
    per = all // k
    nums.sort()
    if nums[-1] > per:
        return False
    n = len(nums)
    dp = [False] * (1 << n)
    dp[0] = True
    cursum = [0] * (1 << n)
    for i in range(0, 1 << n):
        if not dp[i]:
            continue
        for j in range(n):
            if cursum[i] + nums[j] > per:
                break
            if (i >> j & 1) == 0:
                next = i | (1 << j)
                if not dp[next]:
                    cursum[next] = (cursum[i] + nums[j]) % per
                    dp[next] = True
    return dp[(1<<n) - 1]

def ifExist(rootList, num, orgNum):
    if len(rootList) == 0 and num == orgNum:
        return True
    
    flag = False
    for i in range(len(rootList)):
        subList = []
        for j in range(len(rootList)):
            if j != i:
                subList.append(rootList[j])
        if rootList[i] == num:
            flag = flag or ifExist(subList, orgNum, orgNum)
        elif rootList[i] < num:
            flag = flag or ifExist(subList, num - rootList[i], orgNum)
    return flag

def solution(scores):
    sum = 0
    lis = []
    for i in scores:
        sum += i
        lis.append(i)
    for i in range(len(scores), 0, -1):
        if sum % i == 0:
            if ifExist(lis, sum/i, sum/i):
                return sum/i
    return sum

n = int(input())

print(int(solution(nums)))

新学校选址

为了解新学期学生暴涨的问题,小乐村要建立所新学校， 考虑到学生上学安全问题,需要所有学生家到学校的距离最短。 假设学校和所有学生家都走在一条直线之上,请问学校建立在什么位置， 能使得到学校到各个学生家的距离和最短。

输入描述

第一行: 整数 n 取值范围 [1 ,1000 ],表示有 n户家庭。 第二行: 一组整数 m 取值范围 [0, 10000 ] ，表示每户家庭的位置，所有家庭的位置都不相同。

import math

def getResult(arr, n):
    arr.sort()

    if n % 2 == 0:
        mid = int(n / 2)
        return arr[mid - 1]
    else:
        return arr[math.floor(n / 2)]

if __name__ == "__main__":
    n = int(input())
    arr = list(map(int, input().split()))
    print(getResult(arr, n))
