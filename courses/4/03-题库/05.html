上班之路

Jungle 生活在美丽的蓝鲸城，大马路都是方方正正，但是每天马路的封闭情况都不一样。 地图由以下元素组成： 1）”.” — 空地，可以达到; 2）”*” — 路障，不可达到; 3）”S” — Jungle的家; 4）”T” — 公司. 其中我们会限制Jungle拐弯的次数，同时Jungle可以清除给定个数的路障，现在你的任务是计算Jungle是否可以从家里出发到达公司。

offsets = [[-1, 0, 1], [1, 0, 2], [0, -1, 3], [0, 1, 4]]

def dfs(si: int, sj: int, ut: int, uc: int, lastDirect: int, path) -> bool:
    if "T" == matrix[si][sj]:
        return True
    
    for offset in offsets:
        direct = offset[-1]
        newI = si + offset[0]
        newJ = sj + offset[1]
        
        flag1 = False
        flag2 = False

        if 0 <= newI < n and 0 <= newJ < m:
            pos = newI * m + newJ
            if pos in path:
                continue
            if lastDirect != 0 and lastDirect != direct:
                if ut + 1 > t:
                    continue
                flag1 = True
            if "*" == matrix[newI][newJ]:
                if uc + 1 > c:
                    continue
                flag2 = True
            
            path.add(pos)
            res = dfs(newI, newJ, ut + (1 if flag1 else 0), uc + (1 if flag2 else 0), direct, path)
            if res:
                return True
            path.remove(pos)
    return False

def getResult() -> str:
    for i in range(n):
        for j in range(m):
            if "S" == matrix[i][j]:
                path = set([])
                path.add(i * m + j)
                return "YES" if dfs(i, j, 0, 0, 0, path) else "NO"
    return "NO"

if __name__ == "__main__":
    line1 = [int(x) for x in input().split(" ")]
    t, c = line1[0], line1[-1]

    line2 = [int(x) for x in input().split(" ")]
    n, m = line2[0], line2[-1]

    matrix = [["" for _ in range(m)] for _ in range(n)]
    for i in range(n):
        line = input()
        for j in range(m):
            matrix[i][j] = line[j]
    print(getResult())

静态扫描

静态扫描可以快速识别源代码的缺陷，静态扫描的结果以扫描报告作为输出：

1、文件扫描的成本和文件大小相关，如果文件大小为N，则扫描成本为N个金币 2、扫描报告的缓存成本和文件大小无关，每缓存一个报告需要M个金币 3、扫描报告缓存后，后继再碰到该文件则不需要扫描成本，直接获取缓存结果

给出源代码文件标识序列和文件大小序列，求解采用合理的缓存策略，最少需要的金币数。

def StaticScanning(n: int, f: list[int], s: list[int]) -> int:
    count = {}
    size = {}
    for i in range(len(f)):
        k = f[i]
        count.setDefault(k, 0)
        count[k] += 1
        size[k] = s[i]
    ans = 0
    for key, val in count.items():
        ans += min(count[key] * size[key], size[key] + n)
    return ans

if __name__ == "__main__":
    n = int(input())
    f = [int(x) for x in input().split(" ")]
    s = [int(x) for x in input().split(" ")]
    print(StaticScanning(n, f, s))

人数最多的站点

公园园区提供小火车单向通行，从园区站点编号最小到最大通行如1~2~3~4~1， 然后供员工在各个办公园区穿梭， 通过对公司N个员工调研统计到每个员工的坐车区间，包含前后站点， 请设计一个程序计算出小火车在哪个园区站点时人数最多。

def compare(arr1, arr2):
    return arr1[0] - arr2[0]

def MaxSite(n: int, nums: list[list[int]]) -> int:
    nums = sorted(nums, key=functools.cmp_to_key(compare))
    end = []
    hq.heapify(end)
    maxx = 0
    ans = nums[0][0]
    for num in nums:
        s = num[0]
        e = num[1]
        while len(end) != 0:
            top = hq.heappop(end)[1]
            hq.heappush(end, (-top, top))
            if top < s:
                hq.heappop(end)
            else:
                break
        hq.heappush(end, (-e, e))
        if len(end) > maxx:
            maxx = len(end)
            ans = s
    return ans

if __name__ == "__main__":
    n = int(input())
    nums = [[0 for _ in range(2)] for _ in range(n)]
    for i in range(n):
        start, end = map(int, input().split(" "))
        pair = [start, end]
        nums[i] = pair
    print(maxsize(n, nums))

优秀学员统计

公司某部门软件教导团正在组织新员工每日打卡学习活动，他们开展这项学习活动已经一个月了，所以想统计下这个月优秀的打卡员工。

每个员工会对应一个id，每天的打卡记录记录当天打卡员工的id集合，一共30天。

请你实现代码帮助统计出打卡次数top5的员工。假如打卡次数相同，将较早参与打卡的员工排在前面，

如果开始参与打卡的时间还是一样，将id较小的员工排在前面。

注：不考虑并列的情况，按规则返回前5名员工的id即可，如果当月打卡的员工少于5个，按规则排序返回所有有打卡记录的员工id。

import functools

def comp(a, b):
    if a[1] == b[1]:
        if a[2] == b[2]:
            return a[0] - b[0]
        else:
            return a[2] - b[2]
    else:
        return b[1] - a[1]

if __name__ == "__main__":
    n = int(input())
    employee_count = [int(x) for x in input().split(" ")]
    employee_ids = []
    for i in range(30):
        employee_ids.append([int(x) for x in input().split(" ")])
    
    employee_info = {}
    for i in range(30):
        for j in employee_ids[i]:
            if j in employee_info:
                employee_info[j][0] += 1
            else:
                employee_info[j] = [1, i]
    
    employee_list = []
    for key in employee_info:
        employee_list.append([key, employee_info[key][0], employee_info[key][1]])
    
    employee_list = sorted(employee_list, key=functools.cmp_to_key(comp))
    res = []
    for i in range(min(len(employee_list), 5)):
        res.append(str(employee_list[i][0]))
    
    print(" ".join(res))
