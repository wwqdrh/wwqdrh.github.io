枚举所有子字符串情况

计算字符串中的唯一字符个数

然后求和

但是会超过最大值

所以可以根据每个字符的共享值决定，共享值由其左右相同位置的字符位置i、j，可以组合的个数(n - i) * (j - i)

```go
func uniqueLetterString(s string) int {
    idx := map[rune][]int{}
    for i, ch := range s {
        idx[ch] = append(idx[ch], i)
    }
    
    res := 0
    for _, idxs := range idx {
        cur := append(append([]int{-1}, idxs...), len(s))
        for i := 1; i < len(cur) - 1; i++ {
            res += (cur[i] - cur[i-1]) * (cur[i+1] - cur[i])
        }
    }
    return res
}
```
