约瑟夫环模板题

常规模板

f(k)表示k个人中存活下来的下标，就是f(k-1)中存活下来的下标+k

```go
func findTheWinner(n int, k int) int {
    // f(n, k) = (f(n-1, k) + k ) % n
    res := 0
    for i := 1; i <= n; i++ {
        res = (res + k) % i
    }
    return res + 1 // 这里的题目编号从1开始
}
```

这题方向来回变化，但思路是一样的，都是考虑每一步带来的下标变化

```go
func lastRemaining(n int) int {
    if n == 1 {
		return 1
	}
	return 2 * (n / 2 + 1 - lastRemaining(n / 2))
}
```
