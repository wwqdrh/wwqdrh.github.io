预览

number

不管是int、float，在ds里面都是属于number类型

提供+ - * /这些基本操作

var i = 1;
var sum = 0;

while (i <= 100) {
  sum = sum + i;
  i = i + 1;
}

print sum;

string

由双引号和单引号包裹起来，提供+操作符号

print "The " + this.flavor + " cake is " + adjective + "!";

bool

true以及false

var a = true
if a {
    print "a is true"
}

array

由方括号包裹，支持任意类型

支持通过下标访问特定元素

var data = [4,1,2,3,[4,5,6],"hellocloard"];
print data;
var data2 = [4,1,2,3,[4,5,6],"hellocloard"][4];
print data2;
print data[1];
print data[4];

dict

由花括号包裹的key=value数据结构

支持通过key访问特定元素

var data = {a=1};
var data2 = {a=2};
print data;
print data['a'];
print data2['a'];
