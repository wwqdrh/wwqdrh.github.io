## 字符串短链生成
保存长短链的映射

- 自增
- 哈希生成: 选择两个合适的质数(一个用于哈希的数字的进位制，一个控制最大数量)，比如`1117`, `10^9+7`(1g), 当发生冲突时采用线性探测再散列
- 随机生成整数，数量级在`2^32`(4g)

```go
const k1, k2 = 1117, 1e9 + 7

type Codec struct {
    dataBase map[int]string
    urlToKey map[string]int
}

func Constructor() Codec {
    return Codec{map[int]string{}, map[string]int{}}
}

func (c *Codec) encode(longUrl string) string {
    if key, ok := c.urlToKey[longUrl]; ok {
        return "http://tinyurl.com/" + strconv.Itoa(key)
    }
    key, base := 0, 1
    for _, c := range longUrl {
        key = (key + int(c)*base) % k2
        base = (base * k1) % k2
    }
    for c.dataBase[key] != "" {
        key = (key + 1) % k2
    }
    c.dataBase[key] = longUrl
    c.urlToKey[longUrl] = key
    return "http://tinyurl.com/" + strconv.Itoa(key)
}

func (c *Codec) decode(shortUrl string) string {
    i := strings.LastIndexByte(shortUrl, '/')
    key, _ := strconv.Atoi(shortUrl[i+1:])
    return c.dataBase[key]
}
```
# 约瑟夫环

## 常规约瑟夫环

`f(n, k)`

1、n个玩家(玩家编号从1开始)每隔k个(包括起点这一个)就删除一个

`f(n, k) = (f(n-1, k) + k - 1) % n + 1`

```go
func findTheWinner(n, k int) int {
    winner := 1
    for i := 2; i <= n; i++ {
        winner = (k+winner-1)%i + 1
    }
    return winner
}
```

2、n个玩家(玩家编号从1开始)每隔k个(不包括起点这一个)就删除一个

`f(n, k) = (f(n-1, k) + k ) % n + 1`

## 左右往复隔一个元素进行删除

- 定义l(i)为从(1...i)左到右删除剩下的下标
- 定义r(i)为从(1...i)右到左删除剩下的下标

1、由于对称性：l(i) + r(i) = i + 1
2、为了保证有序每次删除后进行下标重建：l(i) = r(i/2) * 2

合并两个公式可得

=> i + 1 - l(i) = l(2i) / 2
=> l(i) = 2 * (i/2 + 1 - l(i/2))

```Go
func lastRemaining(n int) {
	if n == 1 {
		return 1
	}
	return 2 * (n / 2 + 1 - lastRemaining(n / 2))
}
```

## UTF8编码验证

1、如果头字节最高位是0那么当前字符由1个字节组成
2、如果头字节高位为连续的2-4个字节(否则不符合utf8规范)，那么该字符就由连续的1的个数组成，其他字节必须以`10`开头

```go
const mask1, mask2 = 1 << 7, 1 << 7 | 1 << 6

func validUtf8(data []int) bool {
	for index, m := 0, len(data); index < m; {
		n := getBytes(data[index]) // 获取连续的1
		if n < 0 || index+n > m {
			return false
		}
		for _, ch := range data[index+1: index+n] {
			if ch&mask2 != mask1 {
				return false
			}
		}
		index += n
	}
	return true
}

func getBytes(num int) int {
	if num&mask1 == 0 {
		return 1
	}
	
	n := 0
	for mask := mask1; num&mask != 0; mask >>= 1 {
		n++
		if n > 4 {
			return -1
		}
	}
	if n >= 2 {
		return n
	}
	return -1 // 只有一个1不满足
}
```

## 时间差最小的两个时间

1、抽屉原理：这里是处理分钟数，也就是如果`数据>1440`那么就一定存在重复的元素
2、寻找最小的差值：要么使用排序，要么使用桶(其实也算排序了，桶排序)

```go
func getMinutes(t string) int {
	return (int(t[0]-'0')*10 + int(t[1] - '0')) * 60 + int(t[3]-'0')*10 + int(t[4]-'0')
}

func findMinDifference(timePoints []string) int {
	n := len(timePoints)
	if n > 1440 {
		return 0 // 利用抽屉原理剪枝
	}
	sort.Strings(timePoints)
	ans := math.MaxInt32
	
	t0Minutes := getMinutes(timePoints[0])
	preMinutes := t0Minutes
	for _, t := range timePoints[1:] {
		minutes := getMinutes(t)
		ans = min(ans, minutes-preMinutes)
	}
	ans = min(ans, t0Minutes + 1440 - preMinutes) // 首尾时间的时间差
	return ans
}

func min(a, b int) int {
	if a > b {
		return b
	}
	return a
}
```

## 数组结构相同概率取样

直接使用randInt即可

给定一个连续的数组区间，并且包含一个黑名单数据，为了使有效数据连续能够直接使用randInt，可以将数据添加一个映射使其集中化

设n为数据长度，m为黑名单长度, x为`[0,n-m)`中的随机数

-   如果 x 不在黑名单中，则直接返回 x；
-   如果 x 在黑名单中，则返回 x 映射到` [n-m,n)` 范围内的白名单数。

```go
type Solution struct {
	b2w map[int]int
	bound int
}

func Constructor(n int, blacklist []int) Solution {
	m := len(blacklist)
	bound := n - m // 边界
	
	black := map[int]bool{}
	for _, b := range blacklist {
		if b >= bound {
			black[b] = true
		}
	}
	b2w := make(map[int]int, m-len(black)) // 映射
	w := bound
	for _, b := range blacklist {
		if b < bound {
			for black[w] {
				// 寻找一个没有加到black中的数据
				w++
			}
			b2w[b] = w
			w++
		}
	}
	return Solution{b2w, bound}
}

func (s *Solution) Pick() int {
	x := rand.Intn(s.bound)
	if s.b2w[x] > 0 {
		return s.b2w[x]
	}
	return x
}
```

## 在多个矩阵中随机选择整数坐标点

#离散化

由于是整数点，所以每个矩阵坐标点是有限个数，可以将其转换为一维的编号集（同样还需要能够从编号转换到具体在哪个矩阵、哪个坐标），然后在这个编号集中随机选出目标值

通过编号找到属于哪个矩阵哪个下标，依靠`前缀和+二分查找`

```go
type Solution struct {
	rects [][]int // rects[i]表示第i个矩阵, 其中四个值a,b x,y表示左下角与右上角的下标
	sum []int // sum[i]表示前i个矩阵离散化后的下标前缀和
}

func Constructor(rects [][]int) Solution {
	sum := make([]int, len(rects)+1)
	for i, r := range rects {
		a, b, x, y := r[0], r[1], r[2], r[3]
		sum[i+1] = sum[i] + (x-a+1)*(y-b+1) // 前缀和
	}
	return Solution{rects, sum}
}

// 随机选择一个整数下标
// 1、选择编号，2、将编号转换为下标
func (s *Solution) Pick() []int {
	k := rand.Intn(s.sum[len(s.sum)-1]) // [0,n)
	// 使用二分进行搜索，返回k+1在s.sum中可以插入的位置（使用k+1的原因也就是在这，例如遇到相同元素的话也可以保证在其后一个位置）
	rectIndex := sort.SearchInts(s.sum, k+1) - 1
	r := s.rects[rectIndex]
	a, b, y := r[0], r[1], r[3] // y-b+1行数
	// 在当前矩阵中的相对位置
	da := (k - s.sum[rectIndex]) / (y-b+1)
	db := (k - s.sum[rectIndex]) % (y-b+1)
	return []int{a+da, b+db}
}
```



# 坐标系

## 三维坐标中立体结构的投影面积

假设存在x-y-z轴

x-y的投影面积是坐标中值大于0的个数

x-z的投影面积是xi(表示x=i这条直线中最大值)之和

y-z的投影面积是yi(表示y=i这条直线中最大值)之和

```go
func projectArea(grid [][]int) int {
	xy, xz, yz := 0, 0, 0
	for x, row := range grid {
		maxxz, maxyz := 0, 0
		for y, item := range row {
			if item > 0 {
				xy++
			}
			if item > maxxz {
				maxxz = item
			}
			if grid[y][x] > maxyz {
				maxyz = grid[y][x]
			}
		}
		xz += maxxz
		yz += maxyz
	}
	return xy + xz + yz
}
```

## 正方形个数

二维坐标系中，有一系列的点，需要快速找到以某一个点为顶点的正方形的个数（需要满足边平行或垂直与x-y轴）

已知一个点，有了一个x和y坐标，通过遍历同x坐标当前已有的点能够求出正方形的边长，以及其他的两个点

```go
type DetectSquares map[int]map[int]int

func Constructor() DetectSquares {
	return DetectSquares{}
}

func (s DetecSquares) Add(point []int) {
	x, y := point[0], point[1]
	if s[y] == nil {
		s[y] = map[int]int{}
	}
	s[y][x]++
}

func (s DetectSquares) Count(point []int) (ans int) {
	x, y := point[0], point[1]
	if s[y] == nil {
		return
	}
	
	yCnt := s[y] // 当前y坐标有哪些x坐标点以及个数
	for col, colCnt := range s {
		if col != y {
			// 排除同一个点
			d := col - y
			ans += colCnt[x] * colCnt[x+d] * yCnt[x+d]
			ans += colCnt[x] * colCnt[x-d] * yCnt[x-d]
		}
	}
	return
}
```

## 安装栅栏

在二维坐标中有一些树，需要寻找其中能围住所有树的最短的路径

#凸包

`Andrew` 算法正是用于求解凸包上的所有点（围成所有点的最小周长），其算法逻辑将凸包分为「上凸壳」和「下凸壳」

对所有点进行x、y坐标的双关键字排序

分别`从前往后`和`从后往前`处理排序好的所有点

其中vis的作用仅是为了处理凸包第二部分的时候不要使用到第一部分的点

```go
func outerTrees(trees [][]int) [][]int {
	sort.Slice(trees, func(i, j int) bool {
		vala, valb := trees[i], trees[j]
		if vala[0] != valb[0] {
			return vala[0] < valb[0]
		} else {
			return vala[1] < valb[1]
		}
	})
		
	n, tp := len(trees), 0
	stk := make([]int, n+10)
	vis := make([]bool, n+10)
	
	// 从前往后
	tp++ // 不标记起点
	stk[tp] = 0 
	for i := 1; i < n; i++ {
		c := trees[i]
		for tp >= 2 {
			a, b := trees[stk[tp-1]], trees[stk[tp]]
			if getArea(a, b, c) > 0 {
				vis[stk[tp]] = false
				tp--
			} else {
				break
			}
		}
		tp++
		stk[tp] = i
		vis[i] = true
	}
	
	// 从后往前
	size := tp
	for i := n-1; i >= 0; i-- {
		if vis[i] {
			continue
		}
		c := trees[i]
		for tp > size {
			a, b := trees[stk[tp-1]], trees[stk[tp]]
			if getArea(a, b, c) > 0 {
				tp--
			} else {
				break
			}
		}
		tp++
		stk[tp] = i
	}
	
	ans := make([][]int, tp-1)
	for i := range ans {
		ans[i] = make([]int, 2)
	}
	for i := 1; i < tp; i++ {
		ans[i-1] = trees[stk[i]]
	}
	return ans
}

// 向量ab转换为向量ac扫过的面积
func getArea(a, b, c []int) int {
	return cross(subtraction(b, a), subtraction(c, a))
}

// 向量叉乘
func cross(a, b []int) int {
	return a[0] * b[1] - a[1] * b[0]
}

// 坐标转为向量
func subtraction(a, b []int) []int {
	return []int{a[0]-b[0], a[1]-b[1]}
}
```
# 最值
## 在流式数据中保存最大值与最小值

流式数据中存在更新原来数据的情况

> 是在更新的时候就去把堆中的数据同步更新，还是将这个操作分摊到真正查询的时候

```go
type StockPrice struct {
	maxPrice, minPrice *pairHp
	mapping            map[int]int
	maxTimestamp       int
}

type pairHp []pair
type pair struct {
	price, timestamp int
}

func (h pairHp) Len() int              { return len(h) }
func (h pairHp) Less(i, j int) bool    { return h[i].price < h[j].price }
func (h pairHp) Swap(i, j int)         { h[i], h[j] = h[j], h[i] }
func (h *pairHp) Push(val interface{}) { *h = append(*h, val.(pair)) }
func (h *pairHp) Pop() interface{} {
	t := *h
	res := t[len(t)-1]
	*h = t[:len(t)-1]
	return res
}

func (this *StockPrice) Update(timestamp int, price int) {
	heap.Push(this.maxPrice, pair{price: -price, timestamp: timestamp})
	heap.Push(this.minPrice, pair{price: price, timestamp: timestamp})

	if timestamp > this.maxTimestamp {
		this.maxTimestamp = timestamp
	}
	this.mapping[timestamp] = price
}

func (this *StockPrice) Current() int {
	return this.mapping[this.maxTimestamp]
}

func (this *StockPrice) Maximum() int {
	for {
		if p := (*this.maxPrice)[0]; -p.price == this.mapping[p.timestamp] {
			return -p.price
		}
		heap.Pop(this.maxPrice)
	}
}

func (this *StockPrice) Minimum() int {
	for {
		if p := (*this.minPrice)[0]; p.price == this.mapping[p.timestamp] {
			return p.price
		}
		heap.Pop(this.minPrice)
	}
}

func Constructor() StockPrice {
	return StockPrice{
		maxPrice:     new(pairHp),
		minPrice:     new(pairHp),
		maxTimestamp: 0,
		mapping:      map[int]int{},
	}
}
```

```go
func simplifiedFractions(n int) (ans []string) {
    for denominator := 2; denominator <= n; denominator++ {
        for numerator := 1; numerator < denominator; numerator++ {
            if gcd(numerator, denominator) == 1 {
                ans = append(ans, strconv.Itoa(numerator)+"/"+strconv.Itoa(denominator))
            }
        }
    }
    return
}

func gcd(a, b int) int {
	if a < b {
		a, b = b, a
	}
	for b != 0 {
		a, b = b, a % b
	}
	return a
}
```

```go
    }
    if n == 2 {
        return fmt.Sprintf("%d/%d", nums[0], nums[1])
    }
    ans := &strings.Builder{}
    ans.WriteString(fmt.Sprintf("%d/(%d", nums[0], nums[1]))
    for _, num := range nums[2:] {
        ans.WriteByte('/')
        ans.WriteString(strconv.Itoa(num))
    }
    ans.WriteByte(')')
    return ans.String()
}
```

# 排列组合

```go
func countNumbersWithUniqueDigits(n int) int {
	if n == 0 {
		return 1
	}
	if n == 1 {
		return 10
	}
	ans, cur := 10, 9 // 一位数时数字个数，当前可以选择的个数
```

## 求解1-n中满足质数放在质数位的排列种数

需要注意隐藏的信息，排列种类数还由非质数的决定着

```go
const mod int = 1e9 + 7

func numPrimeArrangements(n int) int {
	numPrimes := 0 // 质数个数
	for i := 2; i <= n; i++ {
		if isPrime(i) {
			numPrimes++
		}
	}
	return (factorial(numPrimes) * factorial(n - numPrimes)) % mod
}

func isPrime(n int) bool {
	for i := 2; i*i <= n; i++ {
		if n % i == 0 {
			return false
		}
	}
	return true
}

// n! 阶乘 排列种类数
func factorial(n int) int {
	f := 1
	for i := 1; i <= n; i++ {
		f = (f*i) % mod
	}
	return f
}
```

## 求解字符串表达式结果的种类数

为其添加括号，获得所有的结果

存在重复子问题问题，使用动态规划

自底向上解决问题，最小问题长度为3，然后下一个为5...

`dp[i][j]`表示i, j 区间的所有可能性, 状态转移则是 `op(dp[i][k], dp[k][j])`, 其中op为k位置上的操作数

```go
const addition, subtraction, multiplication = -1, -2, -3

func diffWaysToCompute(expression string) []int {
    ops := []int{}
    for i, n := 0, len(expression); i < n; {
        if unicode.IsDigit(rune(expression[i])) {
            x := 0
            for ; i < n && unicode.IsDigit(rune(expression[i])); i++ {
                x = x*10 + int(expression[i]-'0')
            }
            ops = append(ops, x)
        } else {
            if expression[i] == '+' {
                ops = append(ops, addition)
            } else if expression[i] == '-' {
                ops = append(ops, subtraction)
            } else {
                ops = append(ops, multiplication)
            }
            i++
        }
    }

    n := len(ops)
    dp := make([][][]int, n)
    for i, x := range ops {
        dp[i] = make([][]int, n)
        dp[i][i] = []int{x}
    }
    for sz := 3; sz <= n; sz++ {
        for l, r := 0, sz-1; r < n; l += 2 {
            for k := l + 1; k < r; k += 2 {
                for _, x := range dp[l][k-1] {
                    for _, y := range dp[k+1][r] {
                        if ops[k] == addition {
                            dp[l][r] = append(dp[l][r], x+y)
                        } else if ops[k] == subtraction {
                            dp[l][r] = append(dp[l][r], x-y)
                        } else {
                            dp[l][r] = append(dp[l][r], x*y)
                        }
                    }
                }
            }
            r += 2
        }
    }
    return dp[0][n-1]
}
```
