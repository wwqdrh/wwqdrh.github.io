# 二维数组构造

## 1765、地图最高点

二维矩阵中，grid(i, j)为1就是水域，0就是陆地

需要构造一个二维矩阵，满足以下条件

- 水域高度为0
- 相邻的高度差最多为1
- 高度必须非负

#多源广度优先搜索

使用广度优先搜索，从水域出发，每次更新邻接点并且未计算的地区的高度，满足相邻的最高差为1

```go
type pair struct{ x, y int }
var dirs = []pair{{-1,0}, {1,0}, {0,-1}, {0,1}}

func highestPeak(isWater [][]int) [][]int {
	// m*n的二维切片，使用-1表示还未访问到
	m, n := len(isWater), len(isWater[0])
	ans := make([][]int, m)
	for i := range ans {
		ans[i] = make([]int, n)
		for j := range ans[i] {
			ans[i][j] = -1
		}
	}
	q := []pair{}
	for i, row := range isWater {
		for j, water := range row {
			if water == 1 {
				// 水域入队
				ans[i][j] = 0
				q = append(q, pair{i, j})
			}
		}
	}
	for len(q) > 0 {
		p := q[0]
		q = q[1:]
		for _, d := range dirs {
			if x, y := p.x+d.x, p.y+d.y; x>=0 && x <m && y >=0 && y < n && ans[x][y] == -1 {
				ans[x][y] = ans[p.x][p.y]+1
				q = append(q, pair{x, y})
			}
		}
	}
	return ans
}
```

# 正方形构造

## 给定n条边，判断是否能够组成一个正方形

边可以组合、但是不能拆

#回溯

在回溯的过程中，使用edges来记录4条边已经放入的火柴总长度

由于火柴数n，有4条边的放置情况，即时间复杂度`4^n`

```go
func makesquare(matchsticks []int) bool {
	totalLen := 0
	for _, l := range matchsticks {
		totalLen += 1
	}
	if totalLen%4 != 0 {
		return false
	}
	
	// sort.Reverse返回一个新的接口，改变了Less方法 从大到小
	// sort.Sort(sort.Reverse(sort.IntSlice(matchsticks)))
	sort.Slice(matchsticks, func(i, j int) bool {
		return matchsticks[i] > matchsticks[j]
	})
	edges := [4]int{}
	var dfs func(int) bool
	dfs = func(idx int) bool {
		if idx == len(matchsticks) {
			// 匹配完成了
			// 当idx==n说明，前面的所有火柴都已经放置完成了并且满足每个边都小于等于totalLen/4
			// 由于前面的限制条件，totalLen是能够被四整除，说明正方形已经构成了
			return true
		}
		// 尝试将当前的边放到4条边中，计算之后进行回溯
		for i := range edges {
			edges[i] += matchsticks[idx]
			if edges[i] < totalLen/4 && dfs(idx+1) {
				return true
			}
			edges[i] -= matchsticks[idx]
		}
		return false
	}
	return dfs(0)
}
```
# 数字构造

## 和为k的最少数目(数据集是斐波那契数列)

#贪心

每次选择不超过k的最大斐波那契数

`正确性证明: 证明存在一种选取斐波那契数字数目最少的方案，该方案选取了不超过 k 的最大斐波那契数字`

前提：
- 不存在相邻的元素：不可能选相邻的(因为可以由两个相加的后面一个数替代)
- 不存在重复的元素：各个元素不同(因为如果两个相同的元素例如`f(n)`，可以由`f(n-2) + f(n+1)`进行替代)

反证法：

假设`f(m)`为不超过k的最大斐波那契数字，如果不选，根据上面得出的前提，可以得到可能选取的数之和的最大值N

当m是奇数: `N = F(m-1) + F(m-3) + ... + + F4 + F2` => `F(m-1) + F(m-3) + ... + + F4 + F2 + F1 - F1` => `F(m) - 1` => `< F(m)`

当m是偶数: `N = F(m-1) + F(m-3) + ... + F(3) + F(1)` => `F(m-1) + F(m-3) + ... + F(3) + F(2)` => `F(m)`

所以如果不选择F(m)的话，剩下能够选择的数就是小于等于F(m)的了，所以必须选择F(m)

```go
func findMinFibonacciNumbers(k int) (ans int) {
	f := []int{1, 1}
	for f[len(f)-1] < k {
		// 边构造边选择小于k的最大fibo数
		f = append(f, f[len(f)-1] + f[len(f)-2])
	}
	for i := len(f) - 1; k > 0; i-- {
		if k >= f[i] {
			k -= f[i]
			ans++
		}
	}
	return
}
```

## 连续整数求和

给定一个正整数n，返回连续正整数满足所有数字之和为n的组数

```go
func consecutiveNumbersSum(n int) (ans int) {
	for k := 1; k*(k+1) <= n*2; k++ {
		if isKConsecutive(n, k) {
			ans++
		}
	}
	return
}

// 判断能否由k个连续的数组成n
func isKConsecutive(n, k int) bool {
	if k % 2 == 1 {
		return n % k == 0 // 如果是奇数的话，只要是倍数关系，就可以一个增一个减保证连续关系
	}
	
	return n%k != 0 && 2*n%k == 0
}
```


# 字符串构造
## 满足条件的最长字符串

使用指定个数的a、b、c，构造字符串

1、尽可能长
2、最多有指定的长度
3、不能包含连续的三个相同的元素

#贪心

在遍历过程中，不断排序，选择剩余元素个数最多的字符，如果出现了连续三个相同的且没有其他元素可填了就终止循环

```go
func longestDiverseString(a, b, c int) string {
	ans := []byte{}
	cnt := []struct{ c: int; ch byte }{{a, 'a'}, {b, 'b'}, {c, 'c'}}
	for {
		sort.Slice(cnt, func(i, j int) bool { return cnt[i].c > cnt[j].c })
		hasNext := false
		for i, p := range cnt {
			if p.c <= 0 {
				break
			}
			
			// 判断是否存在三个连续相同的字符
			m := len(ans)
			if m >= 2 && ans[m-2] == p.ch && ans[m-1] == p.ch {
				continue
			}
			hasNext = true
			ans = append(ans, p.ch)
			cnt[i].c--
			break
		}
		if !hasNext {
			return string(ans)
		}
	}
}
```

## 按照指定规则重新生成字符串

`L`表示往左，`R`表示往右，如果方向相对就保持不动

#模拟

```go
func pushDominoes(dominoes string) string {
	s := []byte(dominoes)
	i, n := 0, len(s)
	left, right := byte('L'), byte('R') // left, right表示遍历过程中两边的状态
	
	for i < n {
		j := i
		for j < n && s[j] == '.' {
			j++
		}
		right := byte('R') // 每次循环都要重置下这个最右边的默认方向
		if j < n  {
			right = s[j]
		}
		
		// 判断左右的方向
		if left == right {
			for i < j {
				s[i] = right
				i++
			}
		} else if left == 'R' && right == 'L' {
			k := j - 1
			for i < k {
				s[i] = 'R'
				s[k] = 'L'
				i++
				k--
			}
		}
		left = right
		i = j + 1
	}	
	return string(s)
}
```

```go
// example: PAYPALISHIRING, 3
// P   A   H   N
// A P L S I I G
// Y   I   R
func convert(s string, numRows int) string {
	n, r := len(s), numRows
	if r == 1 || r >= n {
		return s // 如果只有一行或者只能排一列
	}
	
	t := r*2 - 2 // 一个周期
	ans := make([]byte, 0, n)
	for i := 0; i < r; i++ {
		for j := 0; j+i < n; j += t {
			// 每个周期的起始下标
			ans = append(ans, s[j+i]) // 第一个字符
			if i > 0 && i < r-1 && j+t-i < n {
				// 中间的行，并且下一个坐标没有超过n
				ans = append(ans, s[j+t-i])
			}
		}
	}
	return string(ans)
}
```

## 构造最大连续字符串

`TTFF`修改两次，最大连续变成了`TTTT`

#滑动窗口

滑动右窗口直到sum(另一种字符数量)超过k(可以修改的次数)就滑动左窗口

```go
// 分别计算T或者F的最大值
func maxConsecutiveAnswers(answerKey string, k int) int {
    return max(maxConsecutiveChar(answerKey, k, 'T'),
               maxConsecutiveChar(answerKey, k, 'F'))
}

func maxConsecutiveChar(answerKey string, k int, ch byte) (ans int) {
    left, sum := 0, 0
    for right := range answerKey {
        if answerKey[right] != ch {
            sum++
        }
        for sum > k {
            if answerKey[left] != ch {
                sum--
            }
            left++
        }
        ans = max(ans, right-left+1)
    }
    return
}

func max(a, b int) int {
    if b > a {
        return b
    }
    return a
}
```

## 改变字符串中的元素使字符串整体满足规则

#动态规划

### 由01组成的，使得前半部分为0，后半部分为1

设`f(i, 0)`为使得`s[0...i]`子字符串单调递增且最后一个为0的最小翻转次数

设`f(i, 1)`为使得`s[0...i]`子字符串单调递增且最后一个为1的最小翻转次数

*状态转移*

- `f(i, 0) = f(i-1, 0) + (s[i] == 1)`
- `f(i, 1) = min(f(i-1, 0), f(i-1, 1)) + (s[i] == 0)`

由于状态只与前面一个有关，因此可以进行状态优化

```go
func minFlipsMonoIncr(s string) int {
	dp0, dp1 := 0, 0
	for _, ch := range s {
		newdp0, newdp1 := dp0, min(dp0, dp1)
		if ch == '1' {
			newdp0++
		} else {
			newdp1++
		}
		dp0, dp1 = newdp0, newdp1
	}
	return min(dp0, dp1)
}

func min(a, b int) int {
	if a <= b {
		return a
	}
	return b
}
```

# O(1)的最大最小字符串查询

使用双向链表+hash表实现

```go
type node struct {
	keys map[string]struct{}
	count int
}

type AllOne struct {
	*list.List
	nodes map[string]*list.Element
}

func Constructor() AllOne {
	return AllOne{list.New(), map[string]*list.Element{}}
}

func (l *AllOne) Inc(key string) {
    if cur := l.nodes[key]; cur != nil {
        curNode := cur.Value.(node)
        if nxt := cur.Next(); nxt == nil || nxt.Value.(node).count > curNode.count+1 {
            l.nodes[key] = l.InsertAfter(node{map[string]struct{}{key: {}}, curNode.count + 1}, cur)
        } else {
            nxt.Value.(node).keys[key] = struct{}{}
            l.nodes[key] = nxt
        }
        delete(curNode.keys, key)
        if len(curNode.keys) == 0 {
            l.Remove(cur)
        }
    } else { // key 不在链表中
        if l.Front() == nil || l.Front().Value.(node).count > 1 {
            l.nodes[key] = l.PushFront(node{map[string]struct{}{key: {}}, 1})
        } else {
            l.Front().Value.(node).keys[key] = struct{}{}
            l.nodes[key] = l.Front()
        }
    }
}

func (l *AllOne) Dec(key string) {
    cur := l.nodes[key]
    curNode := cur.Value.(node)
    if curNode.count > 1 {
        if pre := cur.Prev(); pre == nil || pre.Value.(node).count < curNode.count-1 {
            l.nodes[key] = l.InsertBefore(node{map[string]struct{}{key: {}}, curNode.count - 1}, cur)
        } else {
            pre.Value.(node).keys[key] = struct{}{}
            l.nodes[key] = pre
        }
    } else { // key 仅出现一次，将其移出 nodes
        delete(l.nodes, key)
    }
    delete(curNode.keys, key)
    if len(curNode.keys) == 0 {
        l.Remove(cur)
    }
}

func (l *AllOne) GetMaxKey() string {
    if b := l.Back(); b != nil {
        for key := range b.Value.(node).keys {
            return key
        }
    }
    return ""
}

func (l *AllOne) GetMinKey() string {
    if f := l.Front(); f != nil {
        for key := range f.Value.(node).keys {
            return key
        }
    }
    return ""
}
```
# 原地修改
## 复写零

在原有数组上，如果遇到0则将后续位置上插入一个0，即再后面的会往后移

#双指针

首先计算原数组中复写之后最后的位置上是哪一个

然后再方向构造出新的数组

```go
func duplicateZeros(arr []int) {
	n := len(arr)
	
	// 计算哪个下标会是最后一个数据
	top := 0 // 记录当前的栈顶
	i := -1
	for top < n {
		i++
		if arr[i] != 0 {
			top++
		} else {
			top += 2
		}
	}
	
	// 构造新的数组数据
	// i指向原来数组下标, j指向新的下标
	j := n-1
	if top == n+1 {
		// 说明最后一个是0 单独拆出来是为了避免多写一个0
		arr[j] = 0
		j--
		i--
	}
	for j >= 0 {
		arr[j] = arr[i]
		j--
		if arr[i] == 0 {
			arr[j] = 0
			j--
		}
		i--
	}
}
```
