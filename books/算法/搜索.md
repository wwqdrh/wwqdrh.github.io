# 路径问题

`m*n`的二维数组中，从点A移动到点B

可能存在的条件

- 存在障碍物不能通过
- 不能走出方格
- 能够移动的方向为上下左右或者8个方向...

## 二维数组元素量级过大

对于元素过多，例如`10^6 * 10^6`的二维数组下。如果直接使用BFS搜索会超时，能够解决的方法有

- 离散化
- 有限步数的广度优先搜索

> 个人认为离散化更优雅，通用性更强

#离散化

不同的行坐标（N+4个）
- 障碍所在的行，最多有 n 个；
- source、target所在的行，最多有 2 个。
- 网格的上下边界（即 -1 和 10^6），有 2 个。

在离散化前，如果两个行坐标本身相邻，那么在离散化之后它们也必须相邻。如果它们不相邻，可以把它们之间间隔的若干行直接「压缩」成一行，即行坐标相差 22。

> 对于列坐标的离散化同理

```go
type pair struct{ x, y int }
var dirs = []pair{
	{-1, 0}, {1, 0}, {0, -1}, {0, 1},
}


func isEscapePossible(block [][]int, source, target []int) bool {
    n := len(block)
    if n < 2 {
        return true
    }
    rows := []int{source[0], target[0]}
    cols := []int{source[1], target[1]}
    for _, b := range block {
        rows = append(rows, b[0])
        cols = append(cols, b[1])
    }

    // 离散化行列坐标
    rMapping, rBound := discrete(rows)
    cMapping, cBound := discrete(cols)

    grid := make([][]bool, rBound+1)
    for i := range grid {
        grid[i] = make([]bool, cBound+1)
    }
    for _, b := range block {
        grid[rMapping[b[0]]][cMapping[b[1]]] = true
    }

    sx, sy := rMapping[source[0]], cMapping[source[1]]
    tx, ty := rMapping[target[0]], cMapping[target[1]]
    grid[sx][sy] = true
    q := []pair{{sx, sy}}
    for len(q) > 0 {
        p := q[0]
        q = q[1:]
        for _, d := range dirs {
            x, y := p.x+d.x, p.y+d.y
            if 0 <= x && x <= rBound && 0 <= y && y <= cBound && !grid[x][y] {
                if x == tx && y == ty {
                    return true
                }
                grid[x][y] = true
                q = append(q, pair{x, y})
            }
        }
    }
    return false
}

func discrete(a []int) (map[int]int, int) {
    sort.Ints(a)

    id := 0
    if a[0] > 0 {
        id = 1
    }
    mapping := map[int]int{a[0]: id}
    pre := a[0]
    for _, v := range a[1:] {
        if v != pre {
            if v == pre+1 {
                id++
            } else {
                id += 2
            }
            mapping[v] = id
            pre = v
        }
    }

    const boundary int = 1e6
    if a[len(a)-1] != boundary-1 {
        id++
    }

    return mapping, id
}
```

## 寻找使得值最大的路径

`m*n`的矩阵，每个单元格都有一个值

- 从任意一个不为0的出发
- 不能进入0的单元格

#回溯

进行dfs选择后，可能不是最大值，递归结构返回上层时记得回溯状态

```go
var dirs = []struct{ x, y int }{
	{-1, 0}, {1, 0}, {0, -1}, {0, 1},
}

func getMaximumGold(grid [][]int) (ans int) {
	var dfs func(x, y, gold int)
	dfs = func(x, y, gold int) {
		gold += grid[x][y]
		if gold > ans {
			ans = gold
		}
		
		rec := grid[x][y]
		grid[x][y] = 0 // 置为0，避免下次重复进入进行开采
		for _, d := range dirs {
			nx, ny := x+d.x, y+d.y
			if nx >= 0 && nx < len(grid) && ny >= 0 && ny < len(grid[nx]) && grid[nx][ny] > 0 {
			// 不超过边界，并且值大于0
				dfs(nx, ny, gold)
			}
		}
		grid[x][y] = rec // 回溯
	}
	
	for i, row := range grid {
		for j, gold := range row {
			if gold > 0 {
				dfs(i, j, 0)
			}
		}
	}
	return
}
```

## 计算走出棋盘的概率

#动态规划

棋盘上，马的移动方向为`(+-(1 | 2), +-(2 | 1))`8个方向

马走k步后仍然留在棋盘上的概率

设`f(step, i, j)`表示马从`i, j`出发走step之后留在棋盘上的概率，状态转移方程为`1/8 * sum(f(step-1, i+c(i), j+c(j)))`, 其中c(i)以及c(j)是当前方向的坐标变化，全部8个方向的状态求和之后除以8就是当前坐标的状态

```go
var dirs = []struct{ i, j int }{
	{-2, -1}, {-2, 1}, {2, -1}, {2, 1}, {-1, -2}, {-1, 2}, {1, -2}, {1, 2},
}

// n*n的棋盘
// k步数
func knightProbaility(n, k, row, column int) float64 {
	dp := make([][][]float64, k+1) // step, i, j: 在i，j坐标开始走step留在棋盘上的概率
	for step := range dp {
		dp[step] = make([][]float64, n)
		for i := 0; i < n; i++ {
			dp[step][i] = make([]float64, n)
			for j := 0; j < n; j++ {
				if step == 0 {
					dp[step][i][j] = 1 // 没有步数还在棋盘上，概率为1
				} else {
					for _, d := range dirs {
						if x, y := i+d.i, j+d.j; 0 <= x && x < n && 0 <= y && y < n {
							dp[step][i][j] += dp[step-1][x][y] / 8
						}
					}
				}
			}
		}
	}
	return dp[k][row][column]
}
```

## 根据坐标进行转换

(x, y) 可以转变为 (x, x + y) 或者 (x+y,  y)

判断两个坐标之间能否相互转换

#dfs

递归搜索，另外添加visit进行剪枝

但是对于 `1 <= sx, sy, tx, ty <= 109`这种量级会直接超出栈容量

```go
type pair struct {
	x, y int
}

// 判断pair1能否转变成pair2
func reachingPoints(sx int, sy int, tx int, ty int) bool {
	target := &pair{x: tx, y: ty}
	visit := map[*pair]bool{}
	var dfs func(*pair) bool
	dfs = func(cur *pair) bool {
		if cur == target {
			return true
		}
		if _, ok := visit[cur]; ok {
			return false
		}
		visit[cur] = true
		
		// dfs路径
		if dfs(&pair{cur.x, cur.x+cur.y}) {
			return true
		}
		if dfs(&pair{cur.x+cur.y, cur.y}) {
			return true
		}
		return false
	}
	return dfs(&pair{x: sx, y: sy})
}
```

#数学规律

如果x=y，不存在上一个状态, 除非x=y=0
如果x>y, 那么上一个状态就是x-y, y
如果x<y, 那么上一个状态就是x,y-x

所以可以通过目标状态进行反向计算

另外不需要x-y x-y ... 一步一步计算，可以直接取模跳过大部分重复的操作

```go
func reachingPoints(sx, sy, tx, ty int) bool {
	for tx > sx && ty > sy && tx != ty {
		if tx > ty {
			tx %= ty
		} else {
			ty %= tx
		}
	}
	
	switch {
	case tx == sx && ty == sy:
		return true
	case tx == sx:
		return ty>sy && (ty-sy)%tx==0
	case ty == sy:
		return tx>sx && (tx-sx)%ty==0
	default:
		return false
	}
}
```

## 寻找字符串变化的最短路径

字符串只有`ACTG`组成，且每次变化都只能变其中一种，并且如果变化不在bank中存在就不是合法的变化

- start: 开始字符串
- end: 目标字符串
- bank: 合法字符串

```go
func minMutation(start, end string, bank []string) int {
    if start == end {
        return 0
    }
    bankSet := map[string]struct{}{}
    for _, s := range bank {
        bankSet[s] = struct{}{}
    }
    if _, ok := bankSet[end]; !ok {
        return -1
    }

    q := []string{start} // bfs 每次将合法的变化都加到步骤中
    for step := 0; q != nil; step++ {
        tmp := q
        q = nil
        for _, cur := range tmp {
            for i, x := range cur {
                for _, y := range "ACGT" {
                    if y != x {
                        nxt := cur[:i] + string(y) + cur[i+1:]
                        if _, ok := bankSet[nxt]; ok {
                            if nxt == end {
                                return step + 1
                            }
                            delete(bankSet, nxt)
                            q = append(q, nxt)
                        }
                    }
                }
            }
        }
    }
    return -1
}
```

# 满足条件的元素个数
## 寻找无法从边界移动到的位置

- 矩阵中的元素由0、1组成
- 只能从1走到1

返回无法走出边界的1（也就是被0包围住，并且不与边界上的1相连）

```go
var dirs = []struct{ x, y int }{
	{-1,0}, {1,0}, {0,-1}, {0, 1},
}

func numEnclaves(grid [][]int) (ans int) {
	// visit数组，保存与边界相连的1
	m, n := len(grid), len(grid[0])
	vis := make([][]bool, m)
	for i := range vis {
		vis[i] = make([]bool, n)
	}
	
	var dfs func(int, int)
	dfs = func(r, c int) {
		if r < 0 || r >= m || c < 0 || c >= n || grid[r][c] == 0|| vis[r][c] {
			// 超出边界、已经访问过、值为0
			return
		}
		vis[r][c] = true
		for _, d := range dirs {
			dfs(r+d.x, c+d.y)
		}
	}
	for i := range grid {
		dfs(i, 0) // 第一列
		dfs(i, n-1) // 最后一列
	}
	for j := 1; j < n-1; j++ {
		dfs(0, j) // 第一行
		dfs(m-1, j) // 最后一行
	}
	
	// 计数
	for i := 1; j < m-1; i++ {
		for j := 1; j < n-1; j++ {
			if grid[i][j] == 1 && !vis[i][j] {
				ans++
			}
		}
	}
	return ans
}
```

## 搜索数组中diff为k的元素

1、数对不能重复
2、两个数不能指向数组中的同一个下标

```go
// 使用visit保存遍历过的元素
// 使用res保存加入到结果的元素, 只保存数对中较小的数(通过这个以及k就能唯一确定大的那个数，所以大的数不需要保存)
func findPairs(nums []int, k int) int {
	visited := map[int]bool{}
	res := map[int]bool{}
	for _, num := range nums {
		if _, ok := visited[num-k]; ok {
			res[num-k] = true
		}
		if _, ok := visited[num+k]; ok {
			res[num] = true
		}
		visited[num] = true
	}
	return len(res)
}
```

```go
func singleNonDuplicate(nums []int) int { 
	low, high := 0, len(nums)-1 
	for low < high { 
		mid := low + (high-low)/2 
		if nums[mid] == nums[mid^1] {
```

```go
func longestWord(words []string) (longest string) {
	sort.Slice(words, func(i, j int) bool {
		s, t := words[i], words[j]
		return len(s) < len(t) || len(s) == len(t) && s > t
	}) // 由短至长，如果相同长度保证后面的字典序更小，使得能够直接更新
	
	candidates := map[string]struct{}{"": {}}
	for _, word := range words {
		if _, ok := candidates[word[:len(word)-1]]; ok {
			longest = word
			candidates[word] = struct{}{}
		}
	}
	return longest
}
```

#状态压缩

```go
func countMaxOrSubsets(nums []int) (ans int) {
	maxOr := 0
	for i := 1; i < 1<<len(nums); i++ {
		or := 0
		for j, num := range nums {
			if i>>j&1 == 1 {
				// 要选择第j个元素
				or |= num
			}
		}
		if or > maxOr {
			maxOr = or
			ans = 1
		} else if or == maxOr {
			ans++
		}
	}
	return
}
```

# TopK问题

> 基于优先队列的算法，都是先加一部分数据到堆中，然后在遍历的过程中在动态的添加更多的元素
## 一维数组的topk

#快速选择

基于快速排序的选择方法：在分解过程中，对子数组进行划分，如果划分得到的q正好就是需要的下标，直接返回a[q], 如果q比目标下标小，就递归右子区间，否则递归左子区间。

这种快速选择变体，使得从原来的递归两个区间变成递归一个区间，直观地理解如果每次规模为 n 的问题我们都划分成 1 和 n - 1，每次递归的时候又向 n - 1 的集合中递归，这种情况是最坏的，时间代价是O(n^2), 可以使用随机化来加速这个过程，时间代价的期望是O(n)

```Go
func findKthLargest(nums []int, k int) int {
    rand.Seed(time.Now().UnixNano())
    return quickSelect(nums, 0, len(nums)-1, len(nums)-k)
}

func quickSelect(a []int, l, r, index int) int {
    q := randomPartition(a, l, r)
    if q == index {
        return a[q]
    } else if q < index {
        return quickSelect(a, q+1, r, index)
    }
    return quickSelect(a, l, q-1, index)
}

func randomPartition(a []int, l, r int) int {
    i := rand.Int() % (r - l + 1)
    a[i], a[r] = a[r], a[i]
    return partition(a, l, r)
}

func partition(a []int, l, r int) int {
    x := a[r]
    i := l - 1
    for j := l; j < r; j++ {
        if a[j] <= x {
            i++
            a[i], a[j] = a[j], a[i]
        }
    }
    a[i+1], a[r] = a[r], a[i+1]
    return i + 1
}
```

#堆

基于大顶堆实现，在面试过程中倾向于自己手写堆排序算法，而不是使用语言自带的库

```Go
func findKthLargest(nums []int, k int) int {
    heapSize := len(nums)
    buildMaxHeap(nums, heapSize)
    for i := len(nums) - 1; i >= len(nums) - k + 1; i-- {
        nums[0], nums[i] = nums[i], nums[0]
        heapSize--
        maxHeapify(nums, 0, heapSize)
    }
    return nums[0]
}

func buildMaxHeap(a []int, heapSize int) {
    for i := heapSize/2; i >= 0; i-- {
        maxHeapify(a, i, heapSize)
    }
}

func maxHeapify(a []int, i, heapSize int) {
    l, r, largest := i*2+1, i*2+2, i
    if l < heapSize && a[l] > a[largest] {
        largest = l
    }
    if r < heapSize && a[r] > a[largest] {
        largest = r
    }
    if largest != i {
        a[i], a[largest] = a[largest], a[i]
        maxHeapify(a, largest, heapSize)
    }
}
```

## 有序矩阵topk

每行和每列元素均按升序排序，找到矩阵中第 `k` 小的元素

矩阵的每一行均为一个有序数组。问题即转化为从这 n 个有序数组中找第 k 大的数，可以想到利用归并排序的做法，归并到第 k 个数即可停止。

```go
type IHeap [][3]int

func (h IHeap) Len() int { return len(h) }
func (h IHeap) Less(i, j int) bool { return h[i][0] < h[j][0] }
func (h IHeap) Swap(i, j int) {
	h[i], h[j] = h[j], h[i]
}

func (h *IHeap) Push(x interface{}) {
	*h = append(*h, x.([3]int))
}

func (h *IHeap) Pop() interface{} {
	old := *h
	n := len(old)
	x := old[n-1]
	*h = old[:n-1]
	return x
}

func kthSmallest(matrix [][]int, k int) int {
	h := &IHeap{}
	for i := 0; i < len(matrix); i++ {
		heap.Push(h, [3]int{matrix[i][0], i, 0})
	}
	
	// 从堆中弹出元素，遍历过程中加入新的元素
	for i := 0; i < k - 1; i++ {
		now := heap.Pop(h).([3]int)
		if now[2] != len(matrix[0]) - 1 {
			heap.Push(h, [3]int{matrix[now[1]][now[2]+1], now[1], now[2]+1})
		}
	}
	return heap.Pop(h).([3]int)[0]
}
```

## 多路数组和topk

```go
type pair struct{ i, j int }

type hp struct {
    data         []pair
    nums1, nums2 []int
}

func (h hp) Len() int { 
	return len(h.data) 
}

func (h hp) Less(i, j int) bool  { 
	a, b := h.data[i], h.data[j]; 
	return h.nums1[a.i]+h.nums2[a.j] < h.nums1[b.i]+h.nums2[b.j] 
}

func (h hp) Swap(i, j int) { 
	h.data[i], h.data[j] = h.data[j], h.data[i] 
}

func (h *hp) Push(v interface{}) { 
	h.data = append(h.data, v.(pair)) 
}

func (h *hp) Pop() interface{}   { 
	a := h.data; 
	v := a[len(a)-1]; 
	h.data = a[:len(a)-1]; 
	return v 
}

func kSmallestPairs(nums1, nums2 []int, k int) (ans [][]int) {
    m, n := len(nums1), len(nums2)
    h := hp{nil, nums1, nums2}
    for i := 0; i < k && i < m; i++ {
        h.data = append(h.data, pair{i, 0})
    }
    for h.Len() > 0 && len(ans) < k {
        p := heap.Pop(&h).(pair)
        i, j := p.i, p.j
        ans = append(ans, []int{nums1[i], nums2[j]})
        if j+1 < n {
            heap.Push(&h, pair{i, j + 1})
        }
    }
    return
}
```

## 特定规则的topk

乘法表中的第k小的数

#归并排序

乘法表的计算规则

`1 * (1 ... n)`
`...`
`n * (1 ... n)`

可以转换为有序矩阵的`topk问题`，不过这里的元素`f(i, j)`不是直接通过下标就能访问，而是通过计算，`(i + 1) * (j + 1)`

```go
// args: 9895 28405 100787757
// returns: 超出时间限制
// description: 在30000*30000的数量级下
type IHeap [][3]int // 值、i、j

func (h IHeap) Len() int { return len(h) }
func (h IHeap) Less(i, j int) bool { return h[i][0] < h[j][0] }
func (h IHeap) Swap(i, j int) {
	h[i], h[j] = h[j], h[i]
}

func (h *IHeap) Push(x interface{}) {
	*h = append(*h, x.([3]int))
}

func (h *IHeap) Pop() interface{} {
	old := *h
	n := len(old)
	x := old[n-1]
	*h = old[:n-1]
	return x
}

func findKthNumber(m int, n int, k int) int {
	if m * n < k {
		// 没有这么多个
		return -1
	}
	h := &IHeap{}
	for i := 0; i < m; i++ {
		heap.Push(h, [3]int{getVal(i, 0), i, 0})
	}
	
	// 从堆中弹出元素，遍历过程中加入新的元素
	for i := 0; i < k - 1; i++ {
		now := heap.Pop(h).([3]int)
		if now[2] != n - 1 {
			heap.Push(h, [3]int{getVal(now[1], now[2]+1), now[1], now[2]+1})
		}
	}
	return heap.Pop(h).([3]int)[0]
}

func getVal(i, j int) int {
	return (i + 1) * (j + 1)
}
```

```go
func findKthNumber(m, n, k int) int {

	return sort.Search(m * n, func(x int) bool {
		count := x / n * n // 上方满的个数
		for i := x / n + 1; i <= m; i++ {
			count += x / i
		}
		return count >= k
	})
}
```

```go
func getSteps(cur, n int) (steps int) {
    first, last := cur, cur

        first *= 10 // 10 100 1000 10000
        last = last*10 + 9 // 19 199

}
```

## 找出第k小的数对距离

在一维数组中，寻找第k小的(a,b)数对, a，b满足`0 <= idx(i) < idx(j) < nums.length`。

#二分查找

第k小的数对差值，最小差值0，最大差值`max(nums) - min(nums)`

二分寻找满足条件的最小距离mid，找到nums中数对差值大于mid的个数cnt，如果`cnt>=目标值k`就算满足条件，即这个mid就是第k小的距离差值

```go
func smallestDistancePair(nums []int, k int) int {
	sort.Ints(nums)
	return sort.Search(nums[len(nums)-1]-nums[0], func(mid int) bool {
		cnt := 0 // 满足数对距离大于mid的个数
		left := 0 // 左区间
		for right, num := range nums {
			for num - nums[left] > mid {
				left++
			}
			cnt += right - left
		}
		return cnt >= k
	})
}
```

# 最大化最小值问题

寻找一个最小值，使得能将数组中的数据分成指定堆数，并且每堆中的元素都小于这个最小值

`O(n*logm)` m是数组中的最大值，每次二分中还需要使用O(n)的复杂度计算总时长是否满足条件

#二分查找 

找到最小以及最大的边界，当满足要求时还需要尝试能否找到满足条件的更小的值

```go
// piles: 数组中的值
// h: 分成的最大堆数
func minEatingSpeed(piles []int, h int) int {
	// 寻找最大速度
	max := 0
	for _, pile := range piles {
		if pile > max {
			max = pile
		}
	}
	
	// sort.Search 0...max-1
	// 寻找满足条件的最小值
	return 1 + sort.Search(max-1, func(speed int) bool {
		speed++ // 1...max
		time := 0
		for _, pile := range piles {
			time += (pile - 1) / speed + 1 // 这一个pile需要的耗时
		}
		return time <= h
	})
}
```
# 滑动窗口

对于线性数据结构，要寻找一个连续的子区间可以使用滑动窗口一般在O(n)的复杂度

## 数字数组寻找连续子数组乘积小于目标值的个数

- 当窗口中乘积小于目标值，滑动右窗口，`窗口值*新值`
- 当窗口中乘积大于目标值，滑动左窗口，`窗口值*旧值`

连续子数组的个数就是`j-i+1`(以j结尾1..2..3..长度的连续子数组)的累加

```go
func numSubarrayProductLessThanK(nums []int, k int) (ans int) {
	prod, i := 1, 0
	for j, num := range nums {
		prod *= num
		for ; i <= j && prod >= k; i++ {
			prod /= nums[i]
		}
		ans += j - i + 1
	}
	return
}
```
