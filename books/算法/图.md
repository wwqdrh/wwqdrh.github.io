# 最小生成树

## prism算法

根据点构建最小生成树，每个点一定都会加入，这是最小生成树的定义决定的，只是找相连的最短路径的短边

```Go
func abs(x, y int) int {
    if x < y {
        return y - x
    }
    return x - y
}

func Prim(points [][]int) int {
    n := len(points)
    // 构建邻接矩阵
    grid := make([][]int, n)
    for i := range grid {
        grid[i] = make([]int, n)
    }
    for i := 0; i < n; i++ {
        for j := 0; j < n; j++ {
            grid[i][j] = abs(points[i][0], points[j][0]) + abs(points[i][1], points[j][1])
        }
    }
    
    // visited记录点的访问状态
    visited := make([]int, n) // 是否已经作为最短路径的边加入
    d := make([]int, n) // 表示与i相连的点中最短的长度
    for i := 1; i < n; i++ {
        d[i] = math.MaxInt32
    }
    
    // 计算最小生成树的长度
    res := 0
    for {
        minCost := math.MaxInt32
        u := -1 // 记录最小的点
        for i := 0; i < n; i++ {
            if visited[i] == 0 && d[i] < minCost {
                minCost = d[i]
                u = i
            }
        }
        if u == -1 {
            // 点都加入了 退出循环
            break
        }
        
        res += minCost
        visited[u] = 1
        for i := 0; i < n; i++ {
            if visited[i] == 0 && grid[u][i] > 0 {
            if grid[u][i] < d[i] {
                d[i] = grid[u][i] // 更新距离数组
            }
            }
        }
    }
    return res
}
```

## Kruskal算法

根据边来选择最小生成树的边，使用并查集来判断是否会造成回路，造成回路的就不会加在结果集中。

```Go
type unionFind struct {
    parent, rank []in
}

func abs(x int) int {
    if x < 0 {
        return -x
    }
    return x
}

func newUnionFind(n int) *unionFind {
    parent := make([]int, n)
    rank := make([]int, n)
    for i := range parent {
        parent[i] = i
        rank[i] = 1
    }
    return &unionFind{parent, rank}
}

func (uf *unionFind) find(x int) int {
    if uf.parent[x] != x {
        uf.parent[x] = uf.find(uf.parent[x])
    }
    return uf.parent[x]
}

func (uf *unionFind) union(x, y int) bool {
    fx, fy := uf.find(x), uf.find(y)
    if fx == fy {
        return false
    }
    if uf.rank[fx] < uf.rank[fy] {
        fx, fy = fy, fx
    }
    uf.rank[fx] += uf.rank[fy]
    uf.parent[fy] = fx
    return true
}

func dist(p, q []int) int {
    return abs(p[0]-q[0]) + abs(p[1]-q[1])
}

func KruskalMinCostConnectPoints(points [][]int) (ans int) {
    n := len(points)
    type edge struct{ v, w, dis int }
    edges := []edge{}
    for i, p := range points {
        for j := i + 1; j < n; j++ {
            edges = append(edges, edge{i, j, dist(p, points[j])})
        }
    }

    sort.Slice(edges, func(i, j int) bool { return edges[i].dis < edges[j].dis })

    uf := newUnionFind(n)
    left := n - 1
    for _, e := range edges {
        if uf.union(e.v, e.w) {
            ans += e.dis
            left--
            if left == 0 {
                break
            }
        }
    }
    return
}
```

# 最短路径

在图中，寻找一个节点到另外一个节点的最短路径

#floyd算法

适用于任何图，有向无向，有权无权

```Go
func floydRoad(graph [][]int) {
    // graph: 表达节点之间的连接关系
    n := len(graph) // 节点的个数
    for k := 0; k < n; k++ {
        for i := 0; i < n; i++ {
            for j := 0; j < n; j++ {
                if graph[i][j] > graph[i][k] + graph[k][j] {
                    graph[i][j] = graph[i][k] + graph[k][j]
                }
            }
        }
    }
}
```

## 跳跃游戏IV

#BFS

给定一个整数数组，需要寻找从开头到结尾最短的路径，跳跃规则包括

- i跳到i+1
- i跳到i-1
- i跳到j，其中满足`arr[i]` == `arr[j]`

数组可以抽象为一个无向图，数组元素为图的顶点，相邻下标的元素之间有一条无向边相连，所有值相同元素之间也有无向边相连。每条边的权重都为 1，即此图为无权图。

```go
func minJumps(arr []int) int {
    n := len(arr)
	// idx保存相同值的是哪些下标
    idx := map[int][]int{}
    for i, v := range arr {
        idx[v] = append(idx[v], i)
    }
	// 保存已经访问过的下标值
    vis := map[int]bool{0: true}
    type pair struct{ idx, step int }
    q := []pair{{}}
    for {
        p := q[0]
        q = q[1:]
        i, step := p.idx, p.step
        if i == n-1 {
			// 找到了
            return step
        }
        for _, j := range idx[arr[i]] {
            if !vis[j] {
                vis[j] = true
                q = append(q, pair{j, step + 1})
            }
        }
		// 及时删除idx中的arr[i], 避免下次遍历到这个值的时候又重新遍历一次
        delete(idx, arr[i])
        if !vis[i+1] {
            vis[i+1] = true
            q = append(q, pair{i + 1, step + 1})
        }
        if i > 0 && !vis[i-1] {
            vis[i-1] = true
            q = append(q, pair{i - 1, step + 1})
        }
    }
}
```

## 求次短路径

无向带权图，需要求解点A到点B的次短路径，另外经过路口的时候可能会由于红灯导致需要等待。

可以记录下所有点与初始点的最短路径与严格次短路径，一旦求得目的点的严格次短路径，我们就可以直接计算到达目的地的第二短时间。

定义`t(i)`为到达i点的时候的时间，`wait(i)`为到达i后准备前往下一个点时候需要等待的时间，`time(i)`为到达`i -> i + 1`所消耗的时间。

那么`t(i+1) = t(i) + wait(i) + time(i)`

```go
// time: 通过任意一条边的时间
// change: 红绿灯交换的时间
func secondMinimum(n int, edges [][]int, time, change int) (ans int) {
	// 构建无向图邻接表
	graph := make([][]int, n + 1)
	for _, e := range edges {
		x, y := e[0], e[1]
		graph[x] = append(graph[x], y)
		graph[y] = append(graph[y], x)
	}
	
	// 设dist(i, 0)表示从1到i的最短路长度
	// 设dist(i, 1)表示从1到i的严格次短长度
	// 初始化为math.MaxInt32
	dist := make([][2]int, n+1)
	dist[1][1] = math.MaxInt32
	for i := 2; i <= n; i++ {
		dist[i] = [2]int{math.MaxInt32, math.MaxInt32}
	}
	
	type pair struct{ x, d int }
	q := []pair{{1, 0}}
	for dist[n][1] == math.MaxInt32 {
		p := q[0]
		q = q[1:]
		for _, y := range graph[p.x] {
			d := p.d + 1
			if d < dist[y][0] {
				// 最小值
				dist[y][0] = d
				q = append(q, pair{y, d})
			} else if dist[y][0] < d && d < dist[y][1] {
				// dist[y][0] < d是因为为了严格大于才行
				dist[y][1] = d
				q = append(q, pair{y, d})
			}
		}
	}
	
	ans := 0
	for i := 0; i < dist[n][1]; i++ {
		// 步数计数
		if ans % (change*2) >= change {
			ans += change*2 - ans%(change*2)
		}
		ans += time
	}
	return ans
}
```
