# 最多可满足的请求

## 最多可达成的换楼请求数目

需要保证每栋楼的动态变化为0

#二进制枚举

```go
func maximumRequests(n int, requests [][]int) (ans int) {
next:
	for mask := 0; mask < 1<<len(requests); mask++ {
		cnt := bits.OnesCount(uint(mask))
		if cnt <= ans {
			continue // 如果当前准备换的请求个数还没有之前的多，直接跳过
		}
		
		delta := make([]int, n) // 记录当前的更换方案下每栋楼的动态变化
		for i, r := range requests {
			if mask>>i & 1 == 1 {
				// 满足当前请求
				delta[r[0]]++
				delta[r[1]]--
			} 
		}
		for _, d := range delta {
			if d != 0 {
				// 不满足
				continue next
			}
		}
		ans = cnt
	}
	return
}
```

# 运行路径

## 求球下降最后会到哪个箱子

```go
func findBall(grid [][]int) []int {
	n := len(grid[0])
	ans := make([]int, n)
	for j := range ans {
		col := j // 球的初始列
		for _, row := range grid {
			dir := row[col] // 当前的挡板方向 1往右走 -1往左走
			col += dir 
			if col < 0 || col == n || row[col] != dir {
				// 到达侧边或者V形
				col = -1
				break
			}
		}
		ans[j] = col
	}
	return ans
}
```
# 对数组操作后返回数组的最小差值

操作数k, 即可以对元素上的数添加`[-k, k]`

找到最大a以及最小值b，如果`a-b-2k > 0`, 说明数组中的元素都可以被`[-k, k]`范围内的数处理成一样的，则结果为0

```go
func smallestRangeI(nums []int, k int) int {
	minNum, maxNum := nums[0], nums[0]
	for _, num := range nums[1:] {
		if num < minNum {
			minNum = num
		}
		if num > maxNum {
			maxNum = num
		}
	}
	ans := maxNum - minNum - 2*k
	if ans > 0 {
		return ans
	}
	return 0
}
```
