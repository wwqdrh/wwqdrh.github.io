#双指针 

使用双指针，`i, j`, 表示当前的字符串，切割后加入res数组，然后合并就行

```go
func reverseWords(s string) string {
    s = strings.TrimSpace(s)
    res := []string{}
    i, j := len(s) - 1, len(s) - 1
    for i >= 0 {
        for i >= 0 && s[i] != ' '{
            i--
        }
        res = append(res, s[i+1:j+1])

        // 跳过中间空格
        for i >= 0 && s[i] == ' ' {
            i--
        }
        j = i
    }
    return strings.Join(res, " ")
}
```

#冒泡排序

> 对一个长度大于等于3的子字符串进行排序，可以拆分成长度为2的字符串的排序子结构

假设`s[t0] == t[0]`, 那么只有`s[0...t0]都大于s[t0]`才能交换到目标位置

我们使用 10 个列表，分别按照从小到大的顺序存储字符 0, 1,⋯,9 在字符串 s 中的位置。在进行遍历的时候，例如`t[i]`, 如果第`t[i]`个列表为空，说明相同的字符不够，无法转换，并且第`[0...t[i]]`的列表要么都为空，要么首位大于`t[i]`, 这样才能移动过去(参考冒泡)

```go
func isTransformable(s string, t string) bool {
    pos := make([][]int, 10)
    for i, ch := range s {
        v, _ := strconv.ParseInt(string(ch), 10, 64)
        pos[v] = append(pos[v], i)
    }

    for _, ch := range t {
        v, _ := strconv.ParseInt(string(ch), 10, 64)
        if len(pos[v]) == 0 {
            return false
        }

        for j := 0; j < int(v); j++ {
            if len(pos[j]) > 0 && pos[j][0] < pos[v][0] {
                return false
            }
        }
        pos[v] = pos[v][1:]
    }

    return true
}
```

给定一个数组，描述了开始字符串到终点字符串，以及沿途所有可选的情况，另外每次只能转换一个字符，寻找这个转换的最短序列，如果不存在则返回0

#图的最短路径

很容易想到图的思路，无向图，将每个字符串抽象成一个节点，如果两个节点之间可以通过一次变化完成那么他们之间存在这样的边。然后就可以基于这个构建的图找到最短路径

其中构建边的情况是

- 最朴素的做法是枚举字符串的每种变化情况，判断是否在字符列表中
- 引入虚拟节点，例如`hit => *it, h*t, hi*`, 与这三个虚拟节点相连，如果其他节点同样的连接到了这个虚拟节点，那么就一定能到`hit`

引入虚拟节点，构造的图会很大，如果只从beginWord开始单向搜索那么效率会很低(因为bfs依赖于每一层的节点个数，参考满二叉树，当然这里不止是二叉，会导致复杂度层指数上升)，可以使用beginWord与endWord双向的bfs搜索

> 无权最短路算法:  Dijkstra / Bellman-Ford / SPFA / Floyd / Edmonds-Karp / Dinic / Prim / Kruskal😭

#二进制枚举

由于长度只有12，所以`2**12`在能够快速处理的范围内

例如: 用 `00` 表示 `a7b`， `01` 表示 `a7B`， `10` 表示 `A7b`

```go
func letterCasePermutation(s string) []string {
	n := len(s)
	bi := 1 << n
	var ans []string
	mp := map[string]struct{}{}
	for i := 0; i < bi; i++ {
		temp := []byte(s)
		for j := range temp {
			if (i>>j)&1 == 1 {
				temp[j] = change(temp[j])
			}
		}
		if _, has := mp[string(temp)]; !has {
			ans = append(ans, string(temp))
			mp[string(temp)] = struct{}{}
		}
	}
	return ans
}
func change(b byte) byte {
	if 'a' <= b && b <= 'z' {
		return 'A' + b - 'a'
	}
	if 'A' <= b && b <= 'Z' {
		return 'a' + b - 'A'
	}
	return b
}
```

5、

二维矩阵，全为01，进行翻转成全0的最小翻转次数

规则: 选择一个位置进行翻转，相邻的同样也会进行翻转

#广度优先搜索

由于该问题数量集只有`3*3`, 可以使用位状态压缩，即512的整数表示二维矩阵每一个情况下的状态。

编写一个二维矩阵向整数的序列化和反序列化，然后常规bfs

```python
class Solution:
    def encode(self, mat, m, n):
        x = 0
        for i in range(m):
            for j in range(n):
                x = x * 2 + mat[i][j]
        return x

    def decode(self, x, m, n):
        mat = [[0] * n for _ in range(m)]
        for i in range(m - 1, -1, -1):
            for j in range(n - 1, -1, -1):
                mat[i][j] = x & 1
                x >>= 1
        return mat

    def convert(self, mat, m, n, i, j):
        for di, dj in [(-1, 0), (1, 0), (0, -1), (0, 1), (0, 0)]:
            i0, j0 = i + di, j + dj
            if 0 <= i0 < m and 0 <= j0 < n:
                mat[i0][j0] ^= 1

    def minFlips(self, mat: List[List[int]]) -> int:
        m, n = len(mat), len(mat[0])
        x_start, step = self.encode(mat, m, n), 0
        if x_start == 0:
            return step

        visited = {x_start}
        q = queue.Queue()
        q.put_nowait(x_start)

        while not q.empty():
            step += 1
            k = q.qsize()
            for _ in range(k):
                status = self.decode(q.get_nowait(), m, n)
                for i in range(m):
                    for j in range(n):
                        self.convert(status, m, n, i, j)
                        x_cur = self.encode(status, m, n)
                        if x_cur == 0:
                            return step
                        if x_cur not in visited:
                            visited.add(x_cur)
                            q.put_nowait(x_cur)
                        self.convert(status, m, n, i, j)

        return -1
```

1、

迷宫中(`# .`两种地形状态)，数组状态整体变换(每走一步变换一次)，可以的操作包括临时变化一个指定的位置，以及永久变化一个指定(每种操作只能执行一次), 在任意时刻进行操作，判断能够在不经过陷阱的情况下到达出口

难点在于何时使用这两次操作，由于只要能到达终点就行，不需要管何时使用这两次操作最优，呢么可以使用递归的思想(分支情况就是当前分支是否使用操作)

```go
var dir4 = []struct{ x, y int }{{-1, 0}, {1, 0}, {0, -1}, {0, 1}}

func escapeMaze(g [][]string) bool {
	k, n, m := len(g), len(g[0]), len(g[0][0])
	vis := make([][][][6]bool, k)
	for i := range vis {
		vis[i] = make([][][6]bool, n)
		for j := range vis[i] {
			vis[i][j] = make([][6]bool, m)
		}
	}
	// s 的最低位：是否使用了临时消除术
	// s 的其余位：0-未使用永久消除术，1-当前正位于永久消除的位置，2-已使用永久消除术
	var f func(t, x, y, s int) bool
	f = func(t, x, y, s int) bool {
		if x < 0 || x >= n || y < 0 || y >= m || t+n-1-x+m-1-y >= k || vis[t][x][y][s] {
			return false
		}
		if x == n-1 && y == m-1 {
			return true
		}
		vis[t][x][y][s] = true
		if s>>1 == 1 {
			for _, d := range dir4 {
				if f(t+1, x+d.x, y+d.y, s^6) { // 跳出永久消除的位置
					return true
				}
			}
			return f(t+1, x, y, s)
		}
		if s>>1 == 0 && g[t][x][y] == '#' && f(t, x, y, s|2) { // 尝试使用永久消除术
			return true
		}
		// 使用永久消除术已无法走到终点，遇到障碍只能使用临时消除术
		if g[t][x][y] == '#' {
			if s&1 == 1 {
				return false
			}
			s |= 1
		}
		for _, d := range dir4 {
			if f(t+1, x+d.x, y+d.y, s) {
				return true
			}
		}
		return f(t+1, x, y, s)
	}
	return f(0, 0, 0, 0)
}
```
