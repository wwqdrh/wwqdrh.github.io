deferиҜӯеҸҘ

з”ЁдәҺиө„жәҗзҡ„йҮҠж”ҫе’ҢејӮеёёзҡ„жҚ•иҺ·гҖӮdeferдёӯзҡ„еҸӮж•°дјҡз«ӢеҚіжү§иЎҢе№¶еӣәе®ҡпјҢеҗҺе…Ҙе…ҲеҮәзҡ„жү§иЎҢйЎәеәҸпјҢdeferдёӯеҸӘжңүеңЁдҝ®ж”№е‘ҪеҗҚиҝ”еӣһеҖјзҡ„ж—¶еҖҷжүҚдјҡеҜ№е…¶дә§з”ҹеҪұе“Қ

еҗҺе…Ҙе…ҲеҮәпјҢdeferеҮҪж•°зҡ„еҸӮж•°еңЁе®ҡд№үзҡ„ж—¶еҖҷе°ұе·Із»ҸзЎ®и®ӨдәҶ

type _defer struct {
		siz     int32 
		started bool
		sp      uintptr // жҢҮеҗ‘ж ҲжҢҮй’Ҳ
		pc      uintptr // жҢҮеҗ‘и°ғз”Ёж–№зҡ„зЁӢеәҸи®Ўж•°еҷЁ
		fn      *funcval // deferе…ій”®еӯ—дёӯдј е…Ҙзҡ„еҮҪж•°
		_panic  *_panic // еҜјиҮҙиҝҗиЎҢdeferзҡ„panic
		link    *_defer // жҳҜжҢҮеҗ‘зҡ„дёҖдёӘ`defer`еҚ•й“ҫиЎЁзҡ„еӨҙпјҢжҜҸж¬Ўе’ұд»¬еЈ°жҳҺдёҖдёӘ`defer`зҡ„ж—¶еҖҷпјҢе°ұдјҡе°ҶиҜҘ`defer`зҡ„ж•°жҚ®жҸ’е…ҘеҲ°иҝҷдёӘеҚ•й“ҫиЎЁеӨҙйғЁзҡ„дҪҚзҪ®
}

deferprocйҖҡиҝҮnewdeferеҲӣе»әдёҖдёӘе»¶иҝҹеҮҪж•°пјҢе№¶е°ҶиҝҷдёӘж–°е»әзҡ„е»¶иҝҹеҮҪж•°жҢӮеңЁеҪ“еүҚgoroutineзҡ„_deferзҡ„й“ҫиЎЁдёҠгҖӮ

newdeferдјҡе…Ҳд»Һschedе’ҢеҪ“еүҚpзҡ„deferpoolеҸ–еҮәдёҖдёӘ_deferз»“жһ„дҪ“пјҢеҰӮжһңdeferpoolжІЎжңү_deferпјҢеҲҷеҲқе§ӢеҢ–дёҖдёӘж–°зҡ„_deferгҖӮ
_deferжҳҜе…іиҒ”еҲ°еҪ“еүҚзҡ„gпјҢжүҖд»ҘdeferеҸӘеҜ№еҪ“еүҚgжңүж•ҲгҖӮ

еңЁйҒҚеҺҶdeferй“ҫиЎЁзҡ„иҝҮзЁӢдёӯпјҢжңүдёӨдёӘйҮҚиҰҒзҡ„з»ҲжӯўжқЎд»¶гҖӮ

  • дёҖдёӘжҳҜеҪ“йҒҚ еҺҶеҲ°й“ҫиЎЁзҡ„жң«е°ҫж—¶пјҢжңҖз»Ҳй“ҫиЎЁжҢҮй’ҲеҸҳдёәnilпјҢиҝҷж—¶йңҖиҰҒз»Ҳжӯўй“ҫиЎЁгҖӮ
  • йҷӨжӯӨ д№ӢеӨ–пјҢеҪ“deferз»“жһ„дёӯеӯҳеӮЁзҡ„SPең°еқҖдёҺеҪ“еүҚdeferreturnзҡ„и°ғз”ЁиҖ…SPең°еқҖдёҚ еҗҢж—¶пјҢд»Қз„¶йңҖиҰҒз»Ҳжӯўжү§иЎҢгҖӮеҺҹеӣ жҳҜеҚҸзЁӢзҡ„й“ҫиЎЁдёӯж”ҫе…ҘдәҶеҪ“еүҚеҮҪж•°и°ғз”Ё й“ҫжүҖжңүеҮҪж•°зҡ„deferз»“жһ„пјҢдҪҶжҳҜеңЁжү§иЎҢж—¶еҸӘиғҪжү§иЎҢеҪ“еүҚеҮҪж•°зҡ„deferз»“ жһ„гҖӮ

еӣ дёәжҳҜдёҖдёӘеҚҸзЁӢдёӯпјҢжүҖд»ҘжүҖжңүеҮҪж•°еҲӣе»әзҡ„deferй“ҫиЎЁжҳҜдёІиЎҢзҡ„пјҢдёҚдјҡеҮәзҺ°еӨҡдёӘеҮҪж•°д№Ӣй—ҙзҡ„deferйЎәеәҸдёҚзЁіе®ҡзҡ„жғ…еҶө
