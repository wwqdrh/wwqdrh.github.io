排查CPU占用过高

top命令确认

go tool pprof http://localhost:6060/debug/pprof/profile

进入交互使用top查看CPU占用较高的调用

top

使用list 名字 查看调用的具体位置

list Eat

如果安装了graphviz工具，使用web命令之后能够在web界面上看到调用链路图

brew install graphviz
