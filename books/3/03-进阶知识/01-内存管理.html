2гҖҒеҮҪж•°дёӯжңҖеҘҪиҝ”еӣһз»“жһ„дҪ“иҖҢдёҚжҳҜжҢҮй’ҲпјҢеӣ дёәиҝ”еӣһжҢҮй’Ҳе®һйҷ…дёҠеңЁеҲҶй…Қзҡ„ж—¶еҖҷжҳҜеңЁе ҶдёҠеҲҶй…ҚеҶ…еӯҳзҡ„пјҢиҷҪз„¶зңҒеҺ»еӨҚеҲ¶зҡ„ж“ҚдҪңдҪҶжҳҜеҚҙдёәgcеёҰжқҘдәҶйә»зғҰ

дёҚеҗҢеҶ…еӯҳжЁЎеһӢеҜ№дәҺиҷҡжӢҹеҶ…еӯҳзҡ„з®ЎзҗҶд»ҘеҸҠдҪҝз”Ёж–№ејҸжҳҜдёҚдёҖж ·зҡ„

goеҶ…еӯҳжЁЎеһӢ

Golang еҶ…еӯҳз®ЎзҗҶжЁЎеһӢдёҺ TCMalloc зҡ„и®ҫи®ЎжһҒе…¶зӣёдјјгҖӮеҹәжң¬иҪ®е»“е’ҢжҰӮеҝөд№ҹеҮ д№ҺзӣёеҗҢпјҢеҸӘжҳҜдёҖдәӣи§„еҲҷе’ҢжөҒзЁӢеӯҳеңЁе·®ејӮгҖӮGolang еҶ…еӯҳз®ЎзҗҶдёӯдҫқз„¶дҝқз•ҷ TCMalloc дёӯзҡ„ PageгҖҒSpanгҖҒSize Class зӯүжҰӮеҝөгҖӮ

  • arean: еҗ‘ж“ҚдҪңзі»з»ҹз”іиҜ·еҶ…еӯҳж—¶зҡ„жңҖе°ҸеҚ•дҪҚпјҢжҜҸдёӘ arena дёә 64MB еӨ§е°Ҹ
  • Page: еҚ•дёӘ arena дјҡиў«еҲҮеҲҶжҲҗд»Ҙ 8KB(64дҪҚзі»з»ҹ) дёәеҚ•дҪҚзҡ„ page
  • mSpan: дёҖдёӘжҲ–еӨҡдёӘpageеҸҜд»Ҙз»„жҲҗдёҖдёӘmSpan, mspan еҸҲеҲҶдёә scan е’Ң noscan дёӨз§ҚпјҢеҲҶеҲ«еҜ№еә”еҶ…йғЁжңүжҢҮй’Ҳзҡ„ object е’ҢеҶ…йғЁжІЎжңүжҢҮй’Ҳзҡ„ objectгҖӮ
  • SizeClass
    • ObjectSize: objectзҡ„еҶ…еӯҳеӨ§е°Ҹ
    • SizeClass: иЎЁзӨәеҲ»еәҰ
    • SpanClass
  • element: mspanеҸҜд»ҘжҢүз…§ sizeclass еҶҚеҲ’еҲҶжҲҗеӨҡдёӘ element

ThreadCache еҒҡдёәжҜҸдёӘзәҝзЁӢзӢ¬з«Ӣзҡ„зј“еӯҳпјҢиғҪеӨҹжҳҺжҳҫзҡ„жҸҗй«ҳ Thread иҺ·еҸ–й«ҳе‘Ҫдёӯзҡ„ж•°жҚ®пјҢе…¶дәҢжҳҜ ThreadCache д№ҹжҳҜд»Һе Ҷз©әй—ҙдёҖж¬ЎжҖ§з”іиҜ·пјҢеҚіеҸӘи§ҰеҸ‘дёҖж¬Ўзі»з»ҹи°ғз”ЁеҚіеҸҜгҖӮ

еҶ…еӯҳеҲҶй…ҚжөҒзЁӢеҸҜйҖҡиҝҮеҜ№иұЎеӨ§е°ҸиҝӣиЎҢеҲҶзұ»

  • TinyеҜ№иұЎпјҢзӣҙжҺҘеңЁMCacheзҡ„TinyиҝӣиЎҢеҲҶй…Қ
  • е°ҸеҜ№иұЎпјҢжҢүз…§span classи§„ж јиҝӣиЎҢеҢ№й…ҚпјҢеҰӮжһңMCacheдёӯзҡ„MSpanжүҖжңүеҶ…еӯҳз©әй—ҙйғҪиў«еҚ з”ЁдәҶпјҢйӮЈд№Ҳе°ұдјҡд»ҺMCentralз”іиҜ·дёҖдёӘеҜ№еә”зҡ„Span
  • еӨ§еҜ№иұЎпјҢзӣҙжҺҘд»ҺMHeapз”іиҜ·йңҖиҰҒзҡ„Page

1B иҮі 16B зҡ„ Tiny еҜ№иұЎ

MCache дёӯдёҚд»…дҝқеӯҳзқҖеҗ„дёӘ Span Class зә§еҲ«зҡ„еҶ…еӯҳеқ—з©әй—ҙпјҢиҝҳжңүдёҖдёӘжҜ”иҫғзү№ж®Ҡзҡ„ Tiny еӯҳеӮЁз©әй—ҙгҖӮTiny з©әй—ҙжҳҜд»Һ Size Class = 2пјҲеҜ№еә” Span Class = 4 жҲ– 5пјүдёӯиҺ·еҸ–дёҖдёӘ 16B зҡ„ ObjectпјҢдҪңдёә Tiny еҜ№иұЎзҡ„еҲҶй…Қз©әй—ҙгҖӮ

Golang еҶ…еӯҳз®ЎзҗҶдёәд»Җд№ҲйңҖиҰҒдёҖдёӘ Tiny иҝҷж ·зҡ„ 16B з©әй—ҙпјҢеҺҹеӣ жҳҜеӣ дёәеҰӮжһңеҚҸзЁӢйҖ»иҫ‘еұӮз”іиҜ·зҡ„еҶ…еӯҳз©әй—ҙе°ҸдәҺзӯүдәҺ 8BпјҢйӮЈд№Ҳж №жҚ®жӯЈеёёзҡ„ Size Class еҢ№й…ҚдјҡеҢ№й…ҚеҲ° Size Class = 1пјҲеҜ№еә” Span Class = 2 жҲ– 3пјүпјҢжүҖд»ҘеғҸ int32гҖҒ byteгҖҒ bool д»ҘеҸҠе°Ҹеӯ—з¬ҰдёІзӯүз»ҸеёёдҪҝз”Ёзҡ„ Tiny еҫ®е°ҸеҜ№иұЎпјҢд№ҹйғҪдјҡдҪҝз”Ёд»Һ Size Class = 1 з”іиҜ·зҡ„иҝҷ 8B зҡ„з©әй—ҙгҖӮдҪҶжҳҜзұ»дјј bool жҲ–иҖ… 1 дёӘеӯ—иҠӮзҡ„ byteпјҢд№ҹйғҪдјҡеҗ„иҮӘзӢ¬дә«иҝҷ 8B зҡ„з©әй—ҙпјҢиҝӣиҖҢеҜјиҮҙжңүдёҖе®ҡзҡ„еҶ…еӯҳз©әй—ҙжөӘиҙ№

жүҖд»Ҙ Golang еҶ…еӯҳз®ЎзҗҶеҶіе®ҡе°ҪйҮҸдёҚдҪҝз”Ё Size Class = 1 зҡ„ SpanпјҢиҖҢжҳҜе°Ҷз”іиҜ·зҡ„ Object е°ҸдәҺ 16B зҡ„з”іиҜ·з»ҹдёҖеҪ’зұ»дёә Tiny еҜ№иұЎз”іиҜ·гҖӮ

16B иҮі 32kB зҡ„е°ҸеҜ№иұЎ

еҲҶй…Қе°ҸеҜ№иұЎзҡ„ж ҮеҮҶжөҒзЁӢжҳҜжҢүз…§ Span Class и§„ж јеҢ№й…Қзҡ„гҖӮ

еӨ§дәҺ32kbзҡ„еӨ§еҜ№иұЎ

е°ҸеҜ№иұЎжҳҜеңЁ MCache дёӯеҲҶй…Қзҡ„пјҢиҖҢеӨ§еҜ№иұЎжҳҜзӣҙжҺҘд»Һ MHeap дёӯеҲҶй…ҚгҖӮеҜ№дәҺдёҚж»Ўи¶і MCache еҲҶй…ҚиҢғеӣҙзҡ„еҜ№иұЎпјҢеқҮжҳҜжҢүз…§еӨ§еҜ№иұЎеҲҶй…ҚжөҒзЁӢеӨ„зҗҶгҖӮ

еӨ§еҜ№иұЎеҲҶй…ҚжөҒзЁӢжҳҜеҚҸзЁӢйҖ»иҫ‘еұӮзӣҙжҺҘеҗ‘ MHeap з”іиҜ·еҜ№иұЎжүҖйңҖиҰҒзҡ„йҖӮеҪ“ PagesпјҢд»ҺиҖҢз»•иҝҮд»Һ MCaceh еҲ° MCentral зҡ„з№Ғзҗҗз”іиҜ·еҶ…еӯҳжөҒзЁӢ
