# 服务发现

## 心跳检测

用于避免服务异常下线，没有去服务中心注销，导致服务中心一直保存着这个服务的信息

`micro --register_ttl=30 --register_interval=15 api`

或者

```go
service := micro.NewService(
	micro.Name("com.example.srv.foo"),
	micro.RegisterTTL(time.Second*30),
	micro.RegisterInterval(time.Second*15),
)
```

服务端启动时注册的命名，启动多个实例会自动将地址加入，供客户端查询到然后进行负载

```go
service := micro.NewService(
		micro.Name("go.micro.srv.greeter"),
	)

```

客户端程序示例

```go
package main

import (
	"encoding/json"
	"log"
	"strings"

	hello "github.com/go-micro/examples/greeter/srv/proto/hello"
	roundrobin "github.com/go-micro/plugins/v4/wrapper/select/roundrobin"
	"go-micro.dev/v4"
	api "go-micro.dev/v4/api/proto"
	"go-micro.dev/v4/errors"

	"context"
)

type Say struct {
	Client hello.SayService
}

func (s *Say) Hello(ctx context.Context, req *api.Request, rsp *api.Response) error {
	log.Print("Received Say.Hello API request")

	name, ok := req.Get["name"]
	if !ok || len(name.Values) == 0 {
		return errors.BadRequest("go.micro.api.greeter", "Name cannot be blank")
	}

	response, err := s.Client.Hello(ctx, &hello.Request{
		Name: strings.Join(name.Values, " "),
	})
	if err != nil {
		return err
	}

	rsp.StatusCode = 200
	b, _ := json.Marshal(map[string]string{
		"message": response.Msg,
	})
	rsp.Body = string(b)

	return nil
}

func main() {
	wrapper := roundrobin.NewClientWrapper()

	service := micro.NewService(
		micro.Name("go.micro.api.greeter"),
		micro.WrapClient(wrapper),
	)

	// parse command line flags
	service.Init()

	service.Server().Handle(
		service.Server().NewHandler(
			&Say{Client: hello.NewSayService("go.micro.srv.greeter", service.Client())},
		),
	)

	if err := service.Run(); err != nil {
		log.Fatal(err)
	}
}
```
