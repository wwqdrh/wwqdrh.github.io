# 名词表

- Dapp	去中心化应用
- DAO	分布式自治组织
- Identity	身份
- Digital identity	数字身份
- Unique identity	特殊身份
- reputation	声誉、满足特定条件的可靠
- escrow	托管
- deposit	保证金
- web of trust	信任网
- Incentive compatibility	激励相容
- collusion	碰撞
- token system	通证系统
- block	区块
- address	地址
- Ether	以太币
- EOA	外部所有账户
- gas	在以太坊区块链中执行的每个操作都需要使用gas支付费用
- gas limit	燃料限制可以应用于单个交易，也可以应用于区块（block-gas-limit，即区块燃料限制）。
- gas price	在一个交易中，对一单位燃料所给出的以太币价格
- transaction	交易
- message	永不序列化，并只在以太坊运行环境中存在的虚拟对象
- Web3	对以太坊平台来说，Web3是一个去中心化app的平台
- epoch	每次再生之间的间隔
- elliptic curve	椭圆曲线，指一种公钥加密的方式
- wallet	指一切可以存储以太币或其他任何加密通证的东西
- contract	包含数据和可执行函数的持久性代码
- Selfdestruct	一个Solidity语言中的全局变量。允许你“毁掉当前的合约，把资金转到给定的地址”。自毁会在区块链上释放空间并避免合约在未来被执行。合约的- 地址会继续存在，但转移到此地址的以太币将会永久消失。
- transaction fee	执行交易
- mining（挖矿）	
- mining pool（矿池）	
- mining reward（挖矿报酬）	
- state（状态）	
- 区块链	一个永远延伸的，由数据区块组成的序列。区块链【当新交易被确认是新区块的一部分时】向后延伸。每个新区块都通过加密PoW链接到已有的区块链
- peer	在网络中其他运行着以太坊节点的，有着区块链完整数据拷贝的计算机
- signing（签名）	
- discovery（peer）	与已知节点交互来发现网络中更多其他节点的过程
- Gas price oracle	Geth客户端的帮助函数（helper function），用来在发送交易时找出一个适当的默认gas价格
- etherbase	
- coinbase	
- balance	账户的加密货币量
- solidity	一种语法类似JavaScript的高级语言。Solidity会被编译为用于EVM的代码。
- serpent	一种语法类似Python的高级语言。会被编译为用于EVM的代码
- EVM	构成以太坊平台核心的分布式计算平台
- distributed hash table（DHT，即分布式hash表）	DHT是提供查询服务的去中心化分布式系统的一个类，与hash表类似，(key, value) 键值对被存储在DHT中，任何参与进来的节点都能高效地检索到与给定key相关联的value
- Nonce（Number Used Once/Number Once，即“只用一次的数字”）	
- Proof-of-Work（PoW，即工作量证明）	
- Proof-of-Stake（PoS，即权益证明）	挖矿的另一种可选方法，需要矿工证明他们拥有一定量的货币。PoS不像PoW那么浪费，但通常情况下PoS和PoW是被共同使用的，以提高网络的安全性
